## Project background

An automotive logistics center needed to stack bins loaded with stamped parts. Each bin is **1.6 m** high; four levels produce a total stack height of **6.4 m**, and each bin carries more than **1 t**. The project therefore required precise docking under high-load, multi-level operating conditions.

## Solution

The project mounted an M4 Pro 3D camera on a smart forklift. The camera estimates the relative pose of the forklift and bin, then provides the offset used to adjust stacking position and angle.

## Implementation points

- **Multi-level operation:** Stack four bins, each with a height of **1.6 m**.
- **Pose-based docking:** Calculate the relative forklift-to-bin pose in real time. Project records report a bin-foot recognition error below **5 mm**.
- **Exception handling:** When the pose offset exceeds the configured range, the system can re-detect, retry docking, or trigger an exception alarm.

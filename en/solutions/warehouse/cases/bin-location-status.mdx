An automated warehouse in South China deployed multiple AGVs to optimize the cargo handling process. However, in the daily operation of the warehouse, manual handling of goods inevitably occurs, causing the storage location occupancy status information obtained by the system to be updated untimely. Information lag may cause the AGV to receive wrong operating instructions, which not only reduces operating efficiency, but may also cause safety risks.

To address this issue, the project uses a bin-location status detection system that monitors every location and sends current status information to the WMS in real time. This enables **digital management of bin locations** and helps AGVs pick and place goods more efficiently, reliably, and safely.

## Solution

### Selection of storage location status recognition sensors

**•Single-point lidar:** Single-point lidar can only emit one laser beam at a time to form a point on the surface of an object. This ranging method can easily ignore the gaps between cartons or pallets, and then identify the current warehouse location as lacking goods or empty, causing stacking accidents.

**•RGB camera:** When detecting the target area through deep learning, once an object outside the training set enters the warehouse, it may cause misdetection and give wrong warehouse location determination information. In addition, when judging warehouse locations with cargo stacking requirements, the information given by the RGB camera lacks cargo height information, making it difficult to arrange stacking tasks.

**•Meier Micro Vision RGB-D Camera:** Can provide three-dimensional data and color information of warehouse locations, with built-in computing power and no need for an external industrial computer.

At the same time, Myer Micro Vision provides an integrated solution of warehouse location status recognition software and hardware, as well as professional model training and deployment guidance and other service support. Considering that the entire solution is cost-effective and easy to deploy, hundreds of sensors were deployed throughout the warehouse to implement comprehensive real-time monitoring of the warehouse location status.

### Detailed introduction of the plan

Meyer Micro Vision's 3D visual warehouse location status identification solution developed based on RGB-D cameras achieves accurate warehouse location identification by collecting 3D point cloud data and RGB data of the warehouse location. It monitors the occupancy status and cargo height information of each warehouse location in real time, provides detailed data support for mobile robots, realizes digital management of warehouse locations, and ensures accurate pick-and-place operations. The storage location identification algorithm is built into the camera, eliminating the need to add external computing units, reducing deployment complexity and reducing project costs.

**•Comprehensive monitoring:** Accurately identify storage location status through three-dimensional data and color information collection, combined with AI technology.

**•Easy deployment:** The storage location status recognition algorithm is placed on the camera side to reduce deployment and maintenance costs.

**•Flexible communication:** Supports multiple communication methods such as TCP/IP, UDP, HTTP, etc., and reports data to the control system in JSON format in real time.

**•Improve efficiency:** Transmit warehouse location information in real time to assist the dispatch system in allocating tasks quickly and accurately.

---
title: "Import and export camera parameters"
description: "Back up and restore obstacle-avoidance, imaging, filter, and pallet-recognition parameters in LxCameraViewer and PalletPro."
---

Export a verified configuration before changing parameters. After importing a configuration, check the camera model, installation extrinsics, and imaging on the target device before operation.

<Warning>
  Configuration files can contain extrinsics that depend on the installation. Do not use one vehicle's extrinsics on another vehicle. Always verify the imported values.
</Warning>

## Obstacle-avoidance parameters

### Export

<Steps>
  <Step title="Open obstacle-avoidance settings">
    In **Application algorithm**, select the obstacle-avoidance algorithm and configure the obstacle-avoidance regions and extrinsics.
  </Step>
  <Step title="Send the parameters">
    In **Obstacle avoidance settings**, select **Send parameters**.
  </Step>
  <Step title="Locate the file">
    Open the `params` folder in the host application directory, then open the directory for the camera ID. The `avoidance` file contains the saved parameters.
  </Step>
</Steps>

The default directory is:

`D:\\Program Files\\Lanxin-MRDVS\\Tools\\params`

### Import

In **Obstacle avoidance settings**, select **Load configuration from file** and choose the exported file. Send the configuration, then verify the region in the point-cloud view and output status.

## Imaging and filter parameters

### Export

1. Configure imaging and filter parameters in **Basic tools**.
2. Stop the camera stream. A configuration cannot be saved while streaming.
3. In **Function settings**, select **Export configuration to file** and create a dedicated folder for the configuration.

### Import

In **Function settings**, select **Load configuration from file** and choose the exported file. Restart the stream and verify image quality and filtering.

## PalletPro pallet parameters

PalletPro `1.4.6` and later can manage pallet-recognition extrinsics in the active algorithm directory.

### Import

<Steps>
  <Step title="Open the camera directory">
    Start **PalletPro**, select **Basic operations → Camera operations → Open directory**, and open the algorithm directory currently used by the camera.
  </Step>
  <Step title="Replace the parameter file">
    Copy or drag the pallet-recognition extrinsics into the directory and confirm replacement of the matching file.
  </Step>
  <Step title="Calibrate again">
    After the import, run pallet docking extrinsic calibration again and verify recognition results.
  </Step>
</Steps>

### Export

After completing and saving calibration, open the active algorithm directory using the same path and back up the pallet-recognition parameter file.

## 3D obstacle avoidance algorithm parameter export and import

1. Parameter export. Under "Application Algorithm", after selecting the corresponding obstacle avoidance algorithm, click on "Obstacle Avoidance Settings". After configuring the camera obstacle avoidance parameters, find "Delivery Parameters" below and click "Execute" to save the parameter file to the corresponding folder.

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/0f405f07-0275-42aa-a152-b1ec4c805eec.avif" alt="D Illustrations related to the export and import of obstacle avoidance algorithm parameters" />
</Frame>

The saved parameters can be viewed in the params folder under the installation path (for example: D:\\Program Files\\Lanxin-MRDVS\\Tools\\params)

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/2a495c06-b9aa-426b-bb59-25077b40f211.avif" alt="D Illustrations related to the export and import of obstacle avoidance algorithm parameters 2" />
</Frame>

## Export and import camera images and filter parameters

1. Parameter export **After configuring the image and filtering parameters in "Basic Tools", **first "stop the flow" of the camera (otherwise the parameters cannot be saved)**, and then click "Execute" after "Export configuration to file" under "Function Settings".

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/7d00bfaf-2a7d-40cd-92a0-31c42f1a1cff.avif" alt="Illustrations related to exporting and importing camera images and filter parameters" />
</Frame>

Create a new folder in the opened directory to save camera parameters (for example, create a new camera parameter folder and save the parameters as test1)

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/e0a29491-8617-4b9c-b284-57ca196f3060.avif" alt="Illustrations related to exporting and importing camera images and filter parameters 2" />
</Frame>

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/5c216209-b509-4237-816d-ec0b3a62e34f.avif" alt="Illustrations related to exporting and importing camera images and filter parameters 3" />
</Frame>

2. Parameter import: Click "Execute" after "Load configuration from file" in the function settings, open the folder where the camera parameters were saved previously and select the required parameter file to open to complete the camera parameter import.

## Interface description

Red menu bar: Click to enter the corresponding function operation details, such as starting mapping, relocation, map management, etc.

Blue box: visible using the loaded 2D contour map and the car positioning display icon

Green box: Displays the real-time RGB image of the top-view camera

Yellow box: real-time display of relocation signal strength (green: excellent strength, blue: good strength, red: poor strength); real-time display of the car's current position coordinates, the car's current movement speed, and the mouse position coordinates

This is the page to start mapping

Then move the AGV to scan the map in the operating area. After the scan is completed, click the menu bar to end the map construction. Please wait patiently (the waiting time is related to the size of the scan area).

When the mapping is successful, the mapping data can be seen in the map management item.

incremental mapping

Function: Add new map areas based on existing maps, such as map expansion and supplement blind spots

Prerequisite: When building incremental maps, you need to ensure that the web page has completed relocation and that you have switched to the map that needs to be added.

(For the time being, manual HTTP request is issued)

Address [http://192.168.100.201:9998/tool/rob/sendCMD](http://192.168.100.201:9998/tool/rob/sendCMD "http://192.168.100.201:9998/tool/rob/sendCMD")

Parameter Json

```json
{

    "cmd":905, // 904 starts incremental mapping; 905 stops incremental mapping

    "attach":{

        "sub_command":0,

        "param":""

}

}
```

## 2. Map management

![2. Illustrations related to map management](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/9b855ff4-d2b8-4a0b-9031-3b1343a848df.avif) Figure 2 Schematic diagram of map management

After the map creation is completed, click **Map Management** on the menu bar. You can see the name of the map you just created in the host map list. Click the Apply button to switch to the map. Similarly, if there are multiple regional maps to switch, you also need to use the map management function to switch. Click the Delete button to delete the corresponding map; **Apply Map:** After clicking Apply, be sure to restart the host before the map will be applied and take effect, otherwise the relocation will not be possible.

## 3. Relocation

After clicking on the relocation, you can drag the mouse to give the approximate relocation pose based on the matching prompt of the laser point cloud and the laser contour map for relocation;

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/2e74ee5f-e192-4fce-8e8e-8f402988dcad.avif" alt="3. Relocation related icons" />
</Frame>

Figure 4 Schematic diagram of relocation function

After the relocation is successful, the **positioning timeout** exception disappears.

QR code relocation

When building a map, you can choose to paste the Apritag code in some locations for relocation. When the complete QR code appears in the device's perspective, it will automatically identify which map and location the code is on, automatically switch the map and relocate.

QR code generation website [https://chaitanyantr.github.io/apriltag](https://chaitanyantr.github.io/apriltag "https://chaitanyantr.github.io/apriltag")

## 7. Firmware update

View package local: Import local firmware that needs to be updated (chassis, actuator, camera firmware).

Restore package: Record the firmware version before the last update. If you need to go back, just click it.

![7. Firmware update related icons](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/db55e5fb-c925-4105-bef1-84ac12f05997.avif) Figure 7 Firmware update

Camera host firmware upgrade

You can use an ordinary host computer to upgrade the firmware. This upgrade method cannot be rolled back and you need to upgrade with caution. ![7. Firmware update related icons 2](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/4ec2ec9c-5285-4a4d-b91e-0fea462d9bed.avif)

## 8. Stop and restart the main application

Click the appropriate control. Before you connect LxCameraViewer to the camera, stop the robot's main application.

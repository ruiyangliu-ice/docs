This page lists the HTTP endpoints provided by the V2 visual localization module. Replace the IP placeholder in each address with the module IP address. Unless an endpoint states otherwise, `code: 200` and `message: success` indicate that the request was processed successfully.

<Warning>
  Updating software, deploying or switching maps, restarting the program, and rebooting the camera interrupt localization. Call these endpoints only while the vehicle is stopped and recovery is available.
</Warning>

## Program and device information

### Update the main program

Address: <code>http://&lt;ip&gt;:9998/tool/rob/quiet_update</code>

<ParamField body="url" type="string" required>
  Reachable URL of the main-program upgrade package.
</ParamField>

### Set camera height

Address: <code>http://&lt;ip&gt;:9998/tool/rob/setHeightRequest</code>

<ParamField body="height" type="number" required>
  Camera height.
</ParamField>

### Get program version

Address: <code>http://&lt;ip&gt;:9998/tool/rob/getRobotVersion</code>

This endpoint requires no request parameters.

<ResponseField name="code" type="integer" required>Response status code.</ResponseField>
<ResponseField name="data" type="string">Program version, such as `version 12313`.</ResponseField>

### Get vehicle information

Address: <code>http://&lt;ip&gt;:9998/tool/rob/getVehicle</code>

This endpoint requires no request parameters.

<ResponseField name="data" type="object" required>Vehicle information object.</ResponseField>
<ResponseField name="vehicleId" type="string">Vehicle UUID.</ResponseField>
<ResponseField name="mapId" type="string">Current map.</ResponseField>
<ResponseField name="vehicleType" type="string">Vehicle type.</ResponseField>

### Get the current pose

Address: <code>http://&lt;ip&gt;:9998/tool/rob/getAGVPose</code>

This endpoint requires no request parameters.

<ResponseField name="pose" type="object" required>Current pose.</ResponseField>
<ResponseField name="x" type="number">X coordinate.</ResponseField>
<ResponseField name="y" type="number">Y coordinate.</ResponseField>
<ResponseField name="yaw" type="number">Heading angle.</ResponseField>

### Relocate

Address: <code>http://&lt;ip&gt;:9998/tool/rob/relocate</code>

Request header: `Content-Type: application/json`

<ParamField body="x" type="number" required>X coordinate in meters.</ParamField>
<ParamField body="y" type="number" required>Y coordinate in meters.</ParamField>
<ParamField body="angle" type="number" required>Pose angle.</ParamField>
<ParamField body="noisyX" type="number" required>X-axis noise in meters.</ParamField>
<ParamField body="noisyY" type="number" required>Y-axis noise in meters.</ParamField>
<ParamField body="noisyAngle" type="number" required>Angular noise.</ParamField>

## Map management

### Download the current running map package

Address: <code>http://&lt;ip&gt;:9998/tool/rob/getRunMapZip</code>

Request header: `Content-Type: application/json`

<ParamField body="Name" type="string" required>Map name.</ParamField>

The response is a ZIP file with content type `application/zip`.

### Deploy a map

Address: <code>http://&lt;ip&gt;:9998/tool/rob/deployRunMap</code>

<ParamField body="id" type="string" required>Map name.</ParamField>

### Apply a map

Address: <code>http://&lt;ip&gt;:9998/tool/rob/changeCurrentRunMap</code>

<ParamField body="name" type="string" required>Map name.</ParamField>

### Start mapping

Address: <code>http://&lt;ip&gt;:9998/tool/rob/startMapping</code>

<ParamField body="name" type="string" required>Map name.</ParamField>

### Stop mapping

Address: <code>http://&lt;ip&gt;:9998/tool/rob/stopMapping</code>

This endpoint requires no request parameters.

### Start or stop incremental mapping

Address: <code>http://&lt;ip&gt;:9998/tool/rob/sendCMD</code>

| Operation | `cmd` | `attach.sub_command` | `attach.param` |
| --- | ---: | ---: | --- |
| Start incremental mapping | `904` | `0` | `mapname` |
| Stop incremental mapping | `905` | `0` | `mapname` |

### Get the map list

Use the local map-list endpoint:

<code>http://&lt;ip&gt;:9998/tool/rob/getRunMapLists</code>

This endpoint requires no request parameters. The response `run_maps` is an array of map names.

<Note>
  The backend map-list endpoint <code>http://&lt;ip&gt;:9998/tool/editor/getLocalMaps</code> is deprecated and not recommended.
</Note>

### Upload a map package to the camera

Address: <code>http://&lt;ip&gt;:9998/tool/rob/uploadMapZip2RunMaps</code>

<ParamField body="url" type="string" required>Reachable URL of the map package.</ParamField>

### Download a map

Address: <code>http://&lt;ip&gt;:9998/tool/rob/downloadVisiomWithLaserMap</code>

<ParamField body="url" type="string" required>Reachable URL of the map package.</ParamField>

## Maintenance endpoints

### Restart the camera program

Address: <code>http://&lt;ip&gt;:9998/tool/rob/restart</code>

This endpoint requires no request parameters.

### Reboot the camera

Address: <code>http://&lt;ip&gt;:9998/tool/rob/Reboot</code>

This endpoint requires no request parameters.

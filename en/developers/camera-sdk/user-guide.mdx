## Overview

This guide is for developers integrating camera devices. It covers the Camera SDK APIs for M4, M4 Mega, M4 Pro, V1 Pro, S2, S2 Max, and H3 Series products. The SDK runs on x86 Windows and x86 or Arm Linux systems.

The Camera SDK provides unified device-access and parameter-configuration APIs. It supports synchronous and asynchronous acquisition of depth, intensity, point cloud, and RGB data. Some camera models also provide obstacle-avoidance, docking, or localization data.

<Info>
  Examples and API descriptions are based on the SDK version shipped with the product. Available parameters may vary by product, firmware, and SDK version.
</Info>

The examples are intended as development guidance and may not cover every device or use case.

1. The default IP network segment of the camera is: 192.168.100.\*

2. On the host computer, set the IP address of the network port connected to the camera to an address in the 192.168.100.x network segment and ensure that it does not conflict with the device. The subnet mask is 255.255.255.0, as shown in the figure below:

##### Linux

1. Unzip the package and run the installation script:

```bash
tar xvf Lanxin-MRDVS-xxx.tar.gz
cd Lanxin-MRDVS
sudo su root
chmod +x install.sh
./install.sh
```

The installation script installs the SDK in `/opt/Lanxin-MRDVS` and sets environment variables for the current user. If the program runs as another user, rerun the script for that user or add the following line to that user's `~/.bashrc`:

```bash
export LD_LIBRARY_PATH=/opt/Lanxin-MRDVS/lib/:$LD_LIBRARY_PATH
```

For terminal sessions that were open before installation, run the following command before starting the program:

```bash
source ~/.bashrc
```

## Calling requirements

### Parameter types and corresponding APIs

1. For enumeration types prefixed with LX\_INT, use DcSetIntValue or DcGetIntValue to set or get INT type parameters;

2. For enumeration types prefixed with LX\_FLOAT, use DcSetFloatValue or DcGetFloatValue to set or get FLOAT type parameters;

3. For enumeration types prefixed with LX\_BOOL, use DcSetBoolValue or DcGetBoolValue to set or get BOOL type parameters;

4. For enumeration types prefixed with LX\_STRING, use DcSetStringValue or DcGetStringValue to set or get string type parameters;

5. Use DcSetCmd to execute instructions for enumeration types prefixed with LX\_CMD;

6. For enumeration types prefixed with LX\_PTR, use DcGetPtrValue to obtain pointer type results;

### API preconditions

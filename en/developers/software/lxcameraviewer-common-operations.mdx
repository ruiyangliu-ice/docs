title: "Common LxCameraViewer operations"

This page summarizes common device configuration and maintenance tasks in LxCameraViewer. For installation, device connection, and the complete interface reference, see the [LxCameraViewer user guide](/en/developers/software/lxcameraviewer).

<Warning>
  Before upgrading camera firmware or an application algorithm, confirm that the package matches the camera model. Do not disconnect power or the network during an upgrade.
</Warning>

## Network and interfaces

### Change the camera IP address

Open **IP configuration tool** from **Other tools**, select the target camera, and update its IP address, subnet mask, or gateway. Keep the host and camera on the same subnet.

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-camera-ip-modification-method.avif" alt="Camera network settings in the IP configuration tool" />
</Frame>

### Configure CAN baud rate and CAN ID

Open **Basic tools → Hardware interface** and change the CAN baud rate or CAN ID. Devices on the same CAN bus must use the same baud rate.

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-modifying-can-baud-rate-and-can-id-methods.avif" alt="CAN baud rate and CAN ID settings in LxCameraViewer" />
</Frame>

## Versions and upgrades

### View software, firmware, and algorithm versions

After opening a camera, view the camera firmware, LxCameraViewer, and application algorithm versions in device information or the software version panel.

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-viewing-methods-of-firmware-version-pc-software-version-and-algorithm-versio.avif" alt="Viewing firmware, software, and algorithm versions in LxCameraViewer" />
</Frame>

### Upgrade camera firmware or an application algorithm

Open **Basic tools → Function settings**, select firmware or application-algorithm upgrade, and choose the delivered upgrade file as prompted.

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-camera-firmware-upgrade-and-application-algorithm-upgrade-methods.avif" alt="Upgrading camera firmware or an application algorithm in LxCameraViewer" />
</Frame>

## Parameters and imaging

### Import or export imaging parameters

Open **Basic tools → Function settings** to export the current camera configuration or load a saved configuration from a file.

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-import-and-export-methods-of-camera-imaging-parameters.avif" alt="Importing and exporting camera imaging parameters in LxCameraViewer" />
</Frame>

### Keep an application algorithm always on

Configure the operating mode in **Application algorithm** so the camera continues outputting the enabled algorithm results after the host application disconnects.

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-applying-algorithm-to-set-normally-open-method.avif" alt="Configuring an application algorithm to stay on in LxCameraViewer" />
</Frame>

### Get camera intrinsics

Open **Basic tools → Function settings** and read the camera intrinsic parameters.

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-the-method-of-obtaining-internal-parameters-of-the-camera.avif" alt="Reading camera intrinsics in LxCameraViewer" />

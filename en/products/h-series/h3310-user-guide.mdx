## Terms and operating principle

### Common terms

| Term | Description |
| --- | --- |
| Baseline | Horizontal distance between the optical centers of the two stereo imaging units. |
| Depth | Data that represents the distance from each pixel to the camera. |
| FOV | Field of view, including horizontal, vertical, and diagonal angles. |
| IR camera | Camera used for near-infrared imaging. |
| RGB camera | Color imaging camera. |
| Depth camera | Imaging system that outputs depth data, typically including projection, imaging, and depth-processing units. |
| SoC | System-on-chip used for image processing, depth calculation, and application algorithms. |

### Binocular coded structured light

H3310 obtains 3D information by combining coded structured-light projection with binocular imaging.

<Steps>
  <Step title="Project the coded pattern">The projector casts an encoded or modulated structured-light pattern onto the target surface.</Step>
  <Step title="Capture synchronously">The two cameras capture the target and projected pattern at the same time.</Step>
  <Step title="Calculate disparity">The system calculates differences and offsets between corresponding patterns in the left and right images.</Step>
  <Step title="Generate depth">The system combines the projection code and binocular disparity to calculate pixel depth and generate a point cloud.</Step>
  <Step title="Measure or localize">Depth and point-cloud data can be used for measurement, localization, recognition, and reconstruction.</Step>
</Steps>

## Product overview

H3310 is an industrial RGB-D structured-light camera weighing 400 g. It can be mounted on a compound robot or small robotic arm. Within a 300–600 mm working range, the camera outputs high-precision depth and RGB data and provides an integrated computing platform for application algorithms.

<Frame>
  <img src="/images/products/h-series-h3310-user-guide-h3310-camera.avif" alt="H3310 camera" />
</Frame>

### Features

- Integrated 6.0 TOPS computing platform for application algorithms.
- Depth, RGB, and amplitude image output.
- Open interfaces for robotic-arm and mobile-robot integration.

## FAQ

<AccordionGroup>
  <Accordion title="The host application cannot open the camera">
    Confirm that the host firewall is not blocking the connection, then check camera power and network connectivity.
  </Accordion>
  <Accordion title="Multiple IP addresses appear in the device list">
    Multiple discoverable devices may be present on the LAN. Select the IP address that belongs to the target camera.
  </Accordion>
  <Accordion title="The data stream is unstable">
    Use a Gigabit Ethernet cable. A 100 Mbps network may cause initial stream instability.
  </Accordion>
  <Accordion title="The software cannot run after installation">
    If the software is installed in a restricted directory, check file and program execution permissions.
  </Accordion>
</AccordionGroup>

<Step title="Power supply">

The camera uses a DC power supply. For signal-wire definitions, see section 3.7, Product interfaces, in the [M4 Pro product user guide](/en/products/m-series/m4-pro-user-guide "M4 Pro product user guide").

</Step>

<Step title="Configure the IP address">

1.  The camera's factory-default IP address is 192.168.100.82. Set the computer to the 192.168.100.XXX subnet.

2.  The camera and computer must be on the same subnet before LxCameraViewer can open the camera.

3.  Disable the network firewall when connecting the camera.

</Step>

<Step title="Configure the computer network">

Open **Control Panel** and select **Network and Internet** → **Network and Sharing Center** → **Change adapter settings** → **Ethernet** → **Internet Protocol Version 4 (TCP/IPv4)**. In the **Internet Protocol Version 4 (TCP/IPv4) Properties** dialog, select **Use the following IP address**. Enter an address on the camera's 192.168.100.XXX subnet and set the subnet mask to 255.255.0.0.

<Frame>
  <img src="/images/products/m-series-m4-pro-quickstart-windows-ipv4-network-settings.avif" alt="Windows IPv4 network settings" />
</Frame>

</Step>

<Step title="Download LxCameraViewer and view images">

1.  Use LxCameraViewer to view images and configure camera image parameters.

2.  Use LxCameraViewer to configure the camera's built-in obstacle-avoidance, pallet, navigation, and grasping algorithms.

3.  Download the LxCameraViewer version for your system. A host application is available for Windows but not Linux. With ROS, run an example and view images in rviz.

- GitHub: [Lanxin-MRDVS/CameraSDK](https://github.com/Lanxin-MRDVS/CameraSDK)

- Software: [Software downloads](/en/resources/downloads/software-and-sdk "Software downloads")

</Step>

<Step title="Start LxCameraViewer">

1.  Double-click the installer to install LxCameraViewer. Then configure the computer's IP address and disable its network firewall.

<Frame>
  <img src="/images/products/m-series-m4-pro-quickstart-lxcameraviewer-interface.avif" alt="LxCameraViewer interface" />
</Frame>

2. Double-click LxCameraViewer and check whether the camera ID appears in the device list on the left.

- If no device appears, check whether the camera indicator flashes every 2s.

- If it flashes normally, troubleshoot the network connection.

- If it remains on without flashing, contact MRDVS technical support.

- If it is off, confirm that the power supply meets the 24V / 3A requirement.

<Frame>
  <img src="/images/products/m-series-m4-pro-quickstart-lxcameraviewer-device-list.avif" alt="LxCameraViewer device list" />
</Frame>

3. Select **Open** to start the stream. The button changes to **Close** when streaming starts. Depth, amplitude, and point-cloud views open by default. To open the RGB view, select its checkbox.

<Frame>
  <img src="/images/products/m-series-m4-pro-quickstart-lxcameraviewer-stream-controls.avif" alt="LxCameraViewer stream controls" />
</Frame>

4. Select the camera parameter settings on the right to expand the available parameters.

<Frame>
  <img src="/images/products/m-series-m4-pro-quickstart-lxcameraviewer-camera-settings.avif" alt="LxCameraViewer camera settings" />
</Frame>

5. Select **Apply algorithm** to choose or switch the application algorithm and select the operating mode.

<Frame>
  <img src="/images/products/m-series-m4-pro-quickstart-lxcameraviewer-algorithm-settings.avif" alt="LxCameraViewer algorithm settings" />

6. These steps cover basic LxCameraViewer operation. Parameters may differ slightly between versions. For current instructions, see the [LxCameraViewer host application user guide](/en/developers/software/lxcameraviewer "LxCameraViewer host application user guide").

</Step>

## Camera precautions

**Follow these instructions. Improper operation may damage internal components.**

1.  Do not drop the camera or subject it to mechanical stress.

2.  Do not modify internal camera files. Doing so may cause permanent damage.

3.  The camera may become warm during extended use, but this does not affect imaging or distance accuracy. It uses passive cooling. Increase the contact area between the camera and its metal mounting plate to improve heat dissipation.

4.  Do not touch the lens. Fingerprints may reduce image quality.

5.  If the camera is not detected, verify that the power supply provides 24V / 3A and that the Ethernet cable is undamaged.

**Camera safety and handling:**

1.  Do not power on the product if you observe external damage.

2.  Do not open any part of the product. Unauthorized disassembly voids after-sales repair service.

3.  Beware of invisible laser radiation. Do not point the camera directly into sunlight.

4.  Do not modify or repair the product. Doing so may violate compliance and safety standards.

The V2 Pro is a navigation controller for mobile-robot and manually operated forklift positioning. It fuses data from an upward-facing camera, 2D LiDAR, and an IMU to support mapping, localization, and relocalization.

## Product overview

### How it works

The controller has an integrated compute platform and does not require an external industrial computer. As the robot moves through the target area, the upward-facing camera captures environmental features, the 2D LiDAR provides planar contours, and the IMU provides motion data. The controller fuses these sources to build a map and output localization results.

### Key features

- Fuses a `108° × 57°` upward-facing camera, a `270°` 2D LiDAR scan, and an IMU.
- Provides `6 TOPS` NPU performance and a visual interface for mapping and map management.
- Integrates Wi-Fi 6 and audio output for manual-forklift positioning data, voice prompts, and alerts.
- Supports indoor positioning and navigation integration for mobile robots, AGVs, and manually operated forklifts.

### Typical applications

- **Factory logistics:** Positioning and navigation for robots in warehouses, loading docks, and workshops.
- **Manual-forklift upgrades:** Add positioning capability to existing forklifts and expose the position data to an upper-layer system.

### Interface definitions

The controller provides power, Ethernet, CAN, I/O, and audio-output interfaces.

<Frame caption="V2 Pro interface locations">
  <img src="/images/products/v-series-v2-pro-overview-v2-pro-interface-definitions.avif" alt="V2 Pro Ethernet, CAN, I/O, power, and audio interfaces" />
</Frame>

| Port | Definition | Description |
| --- | --- | --- |
| CN6 | Ethernet | RJ45 female connector |
| CN7 | CAN | Pink: CAN_H; light green: CAN_L; black: GND |
| CN8 | I/O | Four I/O channels: two inputs and two outputs. See the pinout below. |
| CN9 | Power | 24 VDC / 3 A; 1: positive; G: negative |
| CN10 | Audio output | 3.5 mm audio plug |

#### CN8 I/O pinout

| Pin | Signal | Pin | Signal |
| --- | --- | --- | --- |
| 1 | IN1+ | 2 | IN1− |
| 3 | IN2+ | 4 | IN2− |
| 5 | OUT1+ | 6 | OUT1− |
| 7 | OUT2+ | 8 | OUT2− |

- Inputs: active high at 5–24 V.
- Outputs: push-pull output supporting 24 V / 100 mA drive.

### Wireless communication module

The controller includes a `2T2R` wireless adapter with MU-MIMO and dual-stream transmission. Wi-Fi operates in the `2.400–2.4835 GHz` ISM band. Actual throughput depends on network configuration, site interference, and connected devices.

| Parameter | Specification |
| --- | --- |
| Wi-Fi standard | IEEE 802.11ax/ac/a/b/g/n (2T2R) |
| Bluetooth versions | Bluetooth 5.3, 5.0, 4.2, 4.1, 4.0 LE, 3.0+HS, 2.1+EDR |
| Wi-Fi data rates | 802.11b: 11 Mbps; 802.11a/g: 54 Mbps; 802.11n: MCS0–15; 802.11ac: MCS0–9; 802.11ax: HE0–11 |
| Bluetooth data rates | 1 Mbps, 2 Mbps, up to 3 Mbps |
| Operating band | 2.400–2.4835 GHz ISM band; 5 GHz is not supported |

### Connect and verify the device

<Frame caption="V2 Pro connection test">
  <img src="/images/products/v-series-v2-pro-overview-connection-test.avif" alt="V2 Pro power and Ethernet connection" />
</Frame>

<Steps>
  <Step title="Connect power">
    Connect the power cable to the controller's DC input and connect a `24 V / 3 A` DC supply. A slowly flashing blue indicator indicates normal power.

    If the indicator is off or stays red, check the power connector and adapter.
  </Step>
  <Step title="Configure the network">
    Connect the controller's Ethernet interface directly to the computer with a Cat 6 cable. The factory-default IP is `192.168.100.201`; set the computer to the same subnet and allow device communication through the firewall.

    Open `http://192.168.100.201:9998` in a browser on the LAN to sign in to the web interface without a password.
  </Step>
  <Step title="Check the data stream">
    Start the host application and confirm that the upward-facing camera image is clear and free of artifacts or stutter. If multiple IP addresses are detected, resolve the LAN conflict and select the correct device.
  </Step>
</Steps>

### Interface overview

<Frame caption="V2 Pro host application interface">
  <img src="/images/products/v-series-v2-pro-overview-v2-pro-interface.avif" alt="V2 Pro host application showing the menu, contour map, upward-facing view, and localization status" />
</Frame>

1. **Red area:** Function menu, including mapping, relocalization, and map management.
2. **Blue area:** Loaded 2D contour map and vehicle-localization icon.
3. **Green area:** Live RGB image from the upward-facing camera.
4. **Yellow area:** Relocalization signal strength, vehicle position and speed, and cursor coordinates. Signal colors are green (excellent), blue (good), and red (poor).

### Create a map

<Steps>
  <Step title="Start mapping">
    Select **Start mapping** in the menu.

    <Frame>
      <img src="/images/products/v-series-v2-pro-overview-start-mapping.avif" alt="Start mapping in the V2 Pro host application" />
    </Frame>
  </Step>
  <Step title="Enter a map name">
    A map name may contain letters, numbers, and underscores, but it cannot start with a number.
  </Step>
  <Step title="Capture map data">
    Keep linear speed at or below `0.5 m/s` and angular speed at or below `20°/s`. The interface displays the LiDAR contour and upward-facing camera image.

    <Frame caption="Mapping interface">
      <img src="/images/products/v-series-v2-pro-overview-mapping-interface.avif" alt="V2 Pro mapping interface showing the LiDAR contour and upward-facing image" />
    </Frame>

    Drive slowly and steadily, preferably below `0.2 m/s`. Use straight paths and right-angle turns, avoid repeated paths and random vibration, maintain approximately one-third overlap along paths in the same direction, and keep parallel paths about `2 m` apart. Cover the target area or vehicle route as completely as possible.

    <Frame caption="Recommended mapping path">
      <img src="/images/products/v-series-v2-pro-overview-recommended-mapping-path.avif" alt="Recommended parallel path for V2 Pro mapping" />
    </Frame>
  </Step>
  <Step title="Finish mapping">
    After scanning the area, select **Finish mapping** in the menu. Processing time depends on the scanned area.
  </Step>
</Steps>

### Deploy and apply maps

<Steps>
  <Step title="Deploy the background map">
    Open **Map management**, find the new map in the background-map list, and select **Deploy**. Deploy every map that the host must use; background maps can be edited, exported, or deleted.

    <Frame caption="Background map management">
      <img src="/images/products/v-series-v2-pro-overview-background-map-management.avif" alt="V2 Pro background-map management interface" />
    </Frame>
  </Step>
  <Step title="Apply the host map">
    Select **Apply** in **Host maps**. Restart the controller after applying a map so that it takes effect. Host maps can be exported or deleted.

    <Frame caption="Host map management">
      <img src="/images/products/v-series-v2-pro-overview-host-map-management.avif" alt="V2 Pro host-map management interface" />
    </Frame>
  </Step>
</Steps>

## Troubleshooting

| Problem | Suggested action |
| --- | --- |
| The device does not respond when opened | Check firewall rules and confirm that controller communication is not blocked. |
| The host application detects multiple IP addresses | Resolve the LAN IP conflict and select the correct device address. |
| The stream is unstable | Use a suitable Gigabit Ethernet cable; a lower-speed cable may make initial data transfer unstable. |
| The host application does not start correctly | Install the software in a directory with read and write permissions. |

## Safety and compliance

### Laser safety

<Warning>
The device emits invisible laser radiation during operation. Keep eyes away from the emission area and keep the safety label clear and intact.
</Warning>

- The product uses a Class 1 laser.
- Under normal use, the emitted laser does not present a hazard to people when used according to EN 60825.
- Follow site laser-safety procedures during installation, maintenance, and troubleshooting. Do not disassemble or modify optical components.

<Frame caption="Laser safety label">
  <img src="/images/products/v-series-v2-pro-overview-laser-safety.avif" alt="V2 Pro laser safety label" />
</Frame>

### Contains a lithium battery

<Warning>
The device contains a lithium battery. Damage, short circuits, crushing, puncture, or proximity to fire can cause a fire or other hazards.
</Warning>

- Do not disassemble, modify, or replace the built-in battery yourself.
- Protect the device from severe impact, crushing, puncture, high temperatures, and open flame.
- Follow local requirements when transporting, discarding, or recycling equipment that contains a lithium battery.

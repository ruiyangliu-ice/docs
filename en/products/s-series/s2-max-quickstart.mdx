## First connection

<Steps>
  <Step title="Connect power">
    Connect the supplied 5-pin cable and use a **24 VDC / 2 A** supply. A blue indicator that flashes slowly about every 2 seconds indicates normal power.

    <Frame>
      <img src="/images/products/s-series-s2-max-quickstart-connect-the-s2-max-camera.avif" alt="S2 Max power connection" />
    </Frame>
  </Step>
  <Step title="Connect the network">
    Connect the camera to the computer with a Cat 6 or better Ethernet cable.

    <Frame>
      <img src="/images/products/s-series-s2-max-quickstart-s2-max-ethernet-connection.avif" alt="S2 Max network connection" />
    </Frame>
  </Step>
  <Step title="Configure the IP address">
    The camera's factory-default IP address is `192.168.100.82`. Set the computer to the `192.168.100.xxx` subnet, make sure both devices are on the same subnet, and allow camera communication through the local firewall.
  </Step>
</Steps>

### Windows network settings

In **Control Panel**, open **Network and Internet** → **Network and Sharing Center** → **Change adapter settings** → **Ethernet** → **Internet Protocol Version 4 (TCP/IPv4)**. Select **Use the following IP address**, enter an address on the same subnet, and set the subnet mask to `255.255.0.0`.

## Use LxCameraViewer

### Get the software

LxCameraViewer displays and configures camera image parameters and built-in obstacle-avoidance, pallet, navigation, and grasping algorithms. Windows provides the host application; Linux and ROS can run the examples and use RViz to view data.

- [CameraSDK GitHub repository](https://github.com/Lanxin-MRDVS/CameraSDK)
- [Software downloads](/en/resources/downloads/software-and-sdk)

### Start streaming

<Steps>
  <Step title="Start the software">
    Install and start LxCameraViewer. Confirm that the camera ID appears in the device list.

    - If the device is not listed, check that the indicator is flashing normally and troubleshoot the network.
    - If the indicator stays on, contact MRDVS technical support.
    - If the indicator is off, check the **24 VDC / 2 A** supply.

    <Frame>
      <img src="/images/products/s-series-s2-max-quickstart-lxcameraviewer-interface.avif" alt="LxCameraViewer device list" />
    </Frame>
  </Step>
  <Step title="Open the camera">
    Click **Open** to start streaming. When the button changes to **Close**, streaming has started. Depth, IR amplitude, and point-cloud views are enabled by default; select the corresponding option to enable RGB.

    <Frame>
      <img src="/images/products/s-series-s2-max-quickstart-lxcameraviewer-device-list.avif" alt="LxCameraViewer stream view" />
    </Frame>
  </Step>
  <Step title="Configure parameters and algorithms">
    Expand the right panel to configure camera parameters. Click **Application algorithms** to select an algorithm and operating mode.

    <Frame>
      <img src="/images/products/s-series-s2-max-quickstart-lxcameraviewer-stream-controls.avif" alt="LxCameraViewer stream controls" />
    </Frame>

    <Frame>
      <img src="/images/products/s-series-s2-max-quickstart-lxcameraviewer-camera-parameters.avif" alt="LxCameraViewer camera parameter panel" />
    </Frame>

    <Frame>
      <img src="/images/products/s-series-s2-max-quickstart-lxcameraviewer-application-algorithms.avif" alt="LxCameraViewer application algorithm panel" />
    </Frame>
  </Step>
</Steps>

For more operations, see the [LxCameraViewer host application user guide](/en/developers/software/lxcameraviewer).

## Usage and safety precautions

<Warning>
Incorrect operation can damage internal components. The camera emits invisible laser radiation; do not point the lens at strong sunlight, and do not disassemble, modify, or repair the device yourself.
</Warning>

- Avoid drops, impacts, and mechanical stress.
- Do not modify internal camera files or touch the lens; fingerprints can affect image quality.
- The device may warm during long operation. The camera uses passive cooling; improve heat dissipation with a metal mounting plate.
- If the camera is not detected, verify the **24 VDC / 2 A** supply and check the Ethernet cable.
- Do not power on a visibly damaged camera.

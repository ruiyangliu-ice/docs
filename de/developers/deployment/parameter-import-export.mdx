## Export und Import von Parametern für den 3D-Hindernisvermeidungsalgorithmus

1. Parameterexport. Klicken Sie unter „Anwendungsalgorithmus“ nach Auswahl des entsprechenden Hindernisvermeidungsalgorithmus auf „Hindernisvermeidungseinstellungen“. Nachdem Sie die Parameter zur Hindernisvermeidung der Kamera konfiguriert haben, suchen Sie unten nach „Lieferparameter“ und klicken Sie auf „Ausführen“, um die Parameterdatei im entsprechenden Ordner zu speichern.

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/0f405f07-0275-42aa-a152-b1ec4c805eec.avif" alt="D Abbildungen zum Export und Import von Parametern des Hindernisvermeidungsalgorithmus" />
</Frame>

Die gespeicherten Parameter können im Ordner params unter dem Installationspfad eingesehen werden (zum Beispiel: D:\\Program Files\\Lanxin-MRDVS\\Tools\\params)

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/2a495c06-b9aa-426b-bb59-25077b40f211.avif" alt="D Abbildungen zum Export und Import von Parametern des Hindernisvermeidungsalgorithmus 2" />
</Frame>

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/e95fc782-83fa-4fb2-b44a-4030ec22f948.avif" alt="D Abbildungen zum Export und Import von Parametern des Hindernisvermeidungsalgorithmus 3" />
</Frame>

Suchen Sie im Ordner „params“ nach der Kameranummer. Die darin enthaltene „Vermeidung“ ist der gespeicherte Hindernisvermeidungsparameter.

  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/9a28eb98-ecd5-4333-8245-af2b623e9ab2.avif" alt="D Abbildungen zum Export und Import von Parametern des Hindernisvermeidungsalgorithmus 4" />
</Frame>

</Frame>

  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/f6526600-13be-4b4d-8132-e7e0179b86c5.avif" alt="D Abbildungen zum Export und Import von Parametern des Hindernisvermeidungsalgorithmus 6" />

2. Parameterimport Wenn Sie eine zuvor gespeicherte Konfiguration verwenden möchten, klicken Sie nach „Konfiguration aus Datei laden“ in „Einstellungen zur Hindernisvermeidung“ auf „Ausführen“, um automatisch den Ordner zu öffnen, in dem die Parameter gespeichert sind, und wählen Sie dann die zuvor gespeicherte Parameterdatei zur Hindernisvermeidung für die Konfiguration aus.

## Kamerabilder und Filterparameter exportieren und importieren

1. Parameterexport **Nachdem Sie die Bild- und Filterparameter in „Grundlegende Tools“ konfiguriert haben, **stoppen Sie zunächst den Fluss der Kamera (andernfalls können die Parameter nicht gespeichert werden)** und klicken Sie dann nach „Konfiguration in Datei exportieren“ unter „Funktionseinstellungen“ auf „Ausführen“.

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/7d00bfaf-2a7d-40cd-92a0-31c42f1a1cff.avif" alt="Illustrationen zum Export und Import von Kamerabildern und Filterparametern" />
</Frame>

Erstellen Sie im geöffneten Verzeichnis einen neuen Ordner, um Kameraparameter zu speichern (erstellen Sie beispielsweise einen neuen Kameraparameterordner und speichern Sie die Parameter als test1).

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/e0a29491-8617-4b9c-b284-57ca196f3060.avif" alt="Abbildungen zum Export und Import von Kamerabildern und Filterparametern 2" />
</Frame>

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/5c216209-b509-4237-816d-ec0b3a62e34f.avif" alt="Abbildungen zum Export und Import von Kamerabildern und Filterparametern 3" />
</Frame>

2. Parameterimport: Klicken Sie in den Funktionseinstellungen nach „Konfiguration aus Datei laden“ auf „Ausführen“, öffnen Sie den Ordner, in dem zuvor die Kameraparameter gespeichert wurden, und wählen Sie die gewünschte Parameterdatei zum Öffnen aus, um den Kameraparameterimport abzuschließen.

## 3 Import und Export von Kamera-Tray-Parametern

## 1. Tray-Parameter-Import

**Doppelklicken Sie auf den Desktop von PalletPro, um das Fach zu öffnen**, um die **Software** zu identifizieren (**PalletPro** muss auf die Version **1.4.6** aktualisiert werden)

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/8548ee93-e7fa-4afd-b66e-21e3659030e3.avif" alt="Abbildungen zum Import von Palettenparametern" />
</Frame>

1.1 **Öffnen Sie die Hauptseite und wählen Sie Grundfunktionen aus** → **Kamerabedienung**

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/ec97db6d-4f7b-4b4f-b01f-591adae9332a.avif" alt="Abbildung zum Palettenparameterimport 2" />
</Frame>

1.2 **Öffnen Sie das Verzeichnis und springen Sie zum Algorithmusverzeichnis, das aktuell auf die Kamera angewendet wird**

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/c27f7acc-b681-46a7-a7a2-e9e7747366cf.avif" alt="Abbildung zum Palettenparameterimport 3" />

</Frame>

1.3 **Sie können die erforderlichen externen Parameter für die Palettenerkennung in dieses Verzeichnis einfügen oder die in der Abbildung unten gezeigte Importmethode verwenden, um die externen Parameter für die Palettenerkennung, die zum Ersetzen importiert werden müssen, direkt in dieses Verzeichnis zu ziehen**

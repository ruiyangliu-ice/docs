## Schnittstellenbeschreibung

Rote Menüleiste: Klicken Sie hier, um die entsprechenden Funktionsoperationsdetails einzugeben, z. B. Kartierung starten, verschieben, Kartenverwaltung usw.

Blaues Kästchen: sichtbar anhand der geladenen 2D-Höhenlinienkarte und des Anzeigesymbols für die Fahrzeugpositionierung

Grünes Feld: Zeigt das Echtzeit-RGB-Bild der Draufsichtkamera an

Gelbes Feld: Echtzeitanzeige der Stärke des Verlagerungssignals (grün: ausgezeichnete Stärke, blau: gute Stärke, rot: schlechte Stärke); Echtzeitanzeige der aktuellen Positionskoordinaten des Fahrzeugs, der aktuellen Bewegungsgeschwindigkeit des Fahrzeugs und der Mauspositionskoordinaten

## 1. Beginnen Sie mit dem Erstellen der Karte

Klicken Sie auf der Weboberfläche auf die Schaltfläche **Karte erstellen starten**, wie in der folgenden Abbildung dargestellt:

  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/e400ede7-1ffa-49d1-b62a-e94692079e92.avif" alt="1. Abbildungen zum Starten des Mappings 2" />

Dies ist die Seite, auf der Sie mit der Kartierung beginnen können

Bewegen Sie dann das AGV, um die Karte im Betriebsgebiet zu scannen. Nachdem der Scan abgeschlossen ist, klicken Sie auf die Menüleiste, um die Kartenerstellung zu beenden. Bitte warten Sie geduldig (die Wartezeit hängt von der Größe des Scanbereichs ab).

Wenn die Zuordnung erfolgreich ist, können die Zuordnungsdaten im Kartenverwaltungselement angezeigt werden.

inkrementelle Zuordnung

Funktion: Fügen Sie neue Kartenbereiche basierend auf vorhandenen Karten hinzu, z. B. Kartenerweiterungen und ergänzen Sie tote Winkel

Voraussetzung: Beim Erstellen inkrementeller Karten müssen Sie sicherstellen, dass die Verschiebung der Webseite abgeschlossen ist und Sie zu der Karte gewechselt sind, die hinzugefügt werden muss.

(Derzeit wird eine manuelle HTTP-Anfrage ausgegeben)

Adresse [http://192.168.100.201:9998/tool/rob/sendCMD](http://192.168.100.201:9998/tool/rob/sendCMD "http://192.168.100.201:9998/tool/rob/sendCMD")

Parameter Json

```json
{

    "cmd":905, // 904 starts incremental mapping; 905 stops incremental mapping

    "attach":{

        "sub_command":0,

        "param":""

}

}
```

## 2. Kartenverwaltung

![2. Illustrationen im Zusammenhang mit der Kartenverwaltung](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/9b855ff4-d2b8-4a0b-9031-3b1343a848df.avif) Abbildung 2 Schematische Darstellung der Kartenverwaltung

Nachdem die Kartenerstellung abgeschlossen ist, klicken Sie in der Menüleiste auf **Kartenverwaltung**. Sie können den Namen der Karte, die Sie gerade erstellt haben, in der Host-Kartenliste sehen. Klicken Sie auf die Schaltfläche „Übernehmen“, um zur Karte zu wechseln. Wenn Sie mehrere regionale Karten wechseln möchten, müssen Sie zum Wechseln auch die Kartenverwaltungsfunktion verwenden. Klicken Sie auf die Schaltfläche „Löschen“, um die entsprechende Karte zu löschen. **Karte anwenden:** Stellen Sie nach dem Klicken auf „Übernehmen“ sicher, dass Sie den Host neu starten, bevor die Karte angewendet wird und wirksam wird, andernfalls ist die Verschiebung nicht möglich.

Verschiebung des QR-Codes

Beim Erstellen einer Karte können Sie den Apritag-Code an einigen Stellen zur Verschiebung einfügen. Wenn der vollständige QR-Code in der Perspektive des Geräts angezeigt wird, erkennt es automatisch, auf welcher Karte und an welchem ​​Ort sich der Code befindet, wechselt automatisch die Karte und wechselt den Standort.

Website zur QR-Code-Generierung [https://chaitanyantr.github.io/apriltag](https://chaitanyantr.github.io/apriltag "https://chaitanyantr.github.io/apriltag")

## 7. Firmware-Update

Paket lokal anzeigen: Lokale Firmware importieren, die aktualisiert werden muss (Gehäuse, Aktuator, Kamera-Firmware).

Paket wiederherstellen: Notieren Sie die Firmware-Version vor dem letzten Update. Wenn Sie zurückgehen müssen, klicken Sie einfach darauf.

## 8. Stoppen Sie die Hauptanwendung und starten Sie sie neu

Klicken Sie auf das entsprechende Steuerelement. Bevor Sie LxCameraViewer mit der Kamera verbinden, stoppen Sie die Hauptanwendung des Roboters. Andernfalls behält der Roboter die Kontrolle über die Kamera und verhindert, dass LxCameraViewer eine Verbindung herstellt. ![Hauptsteuerelemente der Anwendung](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/6a00c674-61ac-4cde-8af7-c7e9389ccc6d.avif)

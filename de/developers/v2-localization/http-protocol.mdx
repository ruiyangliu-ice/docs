## Globale Informationen abrufen

### Hauptprogramm aktualisieren

Anfrage-URL

`http://<ip>:9998/tool/rob/quiet_update`

Anfragebeispiel:

```json
{

"url":"http://192.168.2.179:8080/hontai_2.1.20211114.zip"

}
```

### Stellen Sie die Kamerahöhe ein

Anforderungs-URL `http://<ip>:9998/tool/rob/setHeightRequest` Beispiel:

```json
{

"height": 1.3

}
```

Antwortbeispiel:

```json
{

"code":200, "message":"success",

}
```

## Globale Informationen abrufen

### Programmversion abrufen

Anfrage-URL

`http://<ip>:9998/tool/rob/getRobotVersion`

Anforderungsbeispiel: weggelassen

Antwortbeispiel:

```json
{

"code":200, "data":"version 12313", "message":"success"

}
```

### Erhalten Sie grundlegende Fahrzeuginformationen

Anfrage-URL

`http://<ip>:9998/tool/rob/getVehicle`

Anforderungsbeispiel: weggelassen

Antwortbeispiel:

```json
{

"code":200, "message":"success", "data":{

"vehicleId":"vehicle UUID",

"mapId":"current map", "vehicleType":"vehicle model"

}

}
```

### Kartenpaket vom Host abrufen

Anfrage-URL

`http://<ip>:9998/tool/rob/getRunMapZip`

Anforderungsparameter:

Inhaltstyp: application/json

|  |  |  |
| --- | --- | --- |
| Erforderlich | Parameter | Beschreibung |
| Ja | Name | Kartenname |

Antwortbeispiel:

Gibt eine Datei mit dem Inhaltstyp application/zip zurück

### GetAGVPose

Anfrage-URL

`http://<ip>:9998/tool/rob/getAGVPose`

Anforderungsparameter: keine

Antwortbeispiel:

```json
{

    "code": 200,

    "pose": {

        "x": -1.4972912073135376,

        "y": -0.57843643426895142,

        "yaw": 1.6737515926361084

    }

}
```

- 1\. **Umzug (Online gehen)**

Anfrage-URL

`http://<ip>:9998/tool/rob/relocate`

Anforderungsparameter:

Inhaltstyp: application/json

|  |  |  |
| --- | --- | --- |
| Erforderlich | Parameter | Beschreibung |
| Ja | x | X-Position (Meter) |
| Ja | y | Y-Position (Meter) |
| Ja | Winkel | – |
| Ja | noisyX | Laut (Meter) |
| Ja | lautY | Laut (Meter) |
| Ja | noisyAngle | Laut (Meter) |

Anforderungsbeispiel: weggelassen

Antwortbeispiel:

```json
{

"code":200, "message":"success",

}
```

## 相机硬件部分

<AccordionGroup>
<Accordion title="电源指示灯不亮或蓝灯微弱">

**可能原因**

电源电压不稳定

**处理方法**

电源要求24VDC 2A

</Accordion>

<Accordion title="相机频繁自动重启">

**处理方法**

1、检查带宽占用情况<br />2、检查电源供电是否满足

</Accordion>

<Accordion title="上位机软件避障结果正常IO结果异常">

**处理方法**

1、检查IO接线是否正确<br />2、检查功能线是否完好<br />3、检查固件版本是否为5月或之后版本

</Accordion>

<Accordion title="网卡显示断开">

**可能原因**

网络连接异常

**处理方法**

排查网络、网卡、交换机是否正常或是否有拦截

</Accordion>

<Accordion title="canopen协议通讯客户收不到或收到不匹配">

**处理方法**

1、检查波特率、CANID是否匹配<br />2、CAN线连接是否正确

</Accordion>

</AccordionGroup>
## 上位机软件/视觉成像部分

<AccordionGroup>
<Accordion title="上位机软件打开相机后无数据（上位机软件报错未收到数据流）">

**可能原因**

1、防火墙打开；<br />2、网卡与相机网段不匹配；<br />3、交换机拦截；<br />4、客户自身电脑异常导致；

**处理方法**

1、关闭防火墙<br />2、设置同网段IP<br />3、排查交换机问题<br />4、更换电脑<br />5、配置多网段

</Accordion>

<Accordion title="深度图上观测数据缺失标准矩形块，深度呈黑色矩形孔洞">

**可能原因**

多机误检

**处理方法**

停流后在3D设置内关闭多机模式（算法模式需先改为断开时关闭）

</Accordion>

<Accordion title="S10/S10 pro相机软件打开不出图或深度图像紧凑（很小）">

**可能原因**

上位机软件版本不匹配

**处理方法**

更新至Lanxin-MRDVS-1.3.66.0606之后版本

</Accordion>

<Accordion title="相机打开失败，网络通讯错误">

**可能原因**

相机与电脑非同网段

**处理方法**

更改相机或电脑网段至同网段

</Accordion>

<Accordion title="独占应用权限失败">

**可能原因**

多用户连接相机

**处理方法**

确保同一时间只有一个用户连接相机（2.4.50.0720及往后版本会进入多用户权限）

</Accordion>

<Accordion title="受限模式打开相机设备：成功">

**可能原因**

多用户连接相机

**处理方法**

同一时间多个进程连接相机设备，可能出现上位机或其他进程无数据流的情况。需要主权限（第一个打开相机设备的进程）开启数据流后，受限模式设备方可查看数据流。

</Accordion>

<Accordion title="被检测物体成像稀疏或缺失">

**可能原因**

曝光不够或滤波过度过滤

**处理方法**

1、检查高曝光参数（一般在600-1200）<br />2、检查低信号阈值（一般在10-30）

</Accordion>

<Accordion title="上位机启流正常但无图像数据，帧率显示正常">

**可能原因**

激光器微弱或熄灭

**处理方法**

用手机摄像头拍摄相机前脸确认激光器状态，熄灭或微弱则更换相机硬件

</Accordion>

<Accordion title="前方空间无物体，点云上有数据存在">

</Accordion>

<Accordion title="正对被测物时，被测物侧边出现噪声数据">

**可能原因**

眩光导致

**处理方法**

1、打开眩光算法优化<br />2、提高低信号阈值

</Accordion>

<Accordion title="rgb图像中心位置偏红色、rgb图像模糊">

**处理方法**

更换硬件

</Accordion>

<Accordion title="rgb图像正视，深度图像斜视">

**处理方法**

更换硬件

</Accordion>

<Accordion title="相邻物体之间点云存在粘连">

**处理方法**

在滤镜设置中提高去噪级别和平滑级别

</Accordion>

<Accordion title="设置rgbd对齐后rgb无数据">

**可能原因**

软件版本问题

**处理方法**

更新软件版本（Github下载链接：https://github.com/Lanxin-MRDVS/CameraSDK/releases）

</Accordion>

<Accordion title="点云出现明显分层（S系列）">

**处理方法**

更换硬件

</Accordion>

<Accordion title="近处白色物体边缘精度较差（S系列）">

**可能原因**

眩光导致

**处理方法**

打开优化眩光算法，但黑色检测可能缺失

</Accordion>

<Accordion title="反光柱位置大面积过曝（S系列）">

**可能原因**

眩光导致

**处理方法**

打开眩光抑制算法

</Accordion>

<Accordion title="黑色物体成像检测不完整（S系列）">

**可能原因**

打开高曝模式

**处理方法**

TOF寄存器100130B0设置为240

</Accordion>

</AccordionGroup>

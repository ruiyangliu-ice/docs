## 常见问题

<AccordionGroup>
<Accordion title="黑托盘托盘识别失败">

**可能原因**

黑色托盘成像不完整

**处理方法**

1、上位机软件中“3D设置”内“高曝光”修改为1400；<br />2、“滤波”内“低信号阈值”设置为20；<br />3、“滤波”内“小信号检测”设置为1

</Accordion>

<Accordion title="palletpro检测正常但车体收到检测结果异常或失败">

**可能原因**

软件与嵌入式版本不统一

**处理方法**

更新嵌入式算法固件

</Accordion>

<Accordion title="外参标定报错显示未检测到地面">

**可能原因**

地面点云未成像

**处理方法**

上位机软件打开相机，3D设置内高曝光修改为1400，滤波设置内低信号阈值设置为10，进行外参标定，标定完成后修改回原参数

</Accordion>

<Accordion title="近端外参标定失败">

**处理方法**

1、检查托盘成像是否完整<br />2、托盘与相机距离是否在1300mm左右

</Accordion>

<Accordion title="托盘识别结果返回failed-1或-11">

**可能原因**

无数据或数据不足

**处理方法**

通过上位机软件确认相机是否正常出图，或是否滤波较大过滤掉较多点云

</Accordion>

<Accordion title="托盘识别结果返回failed-2或或-12或-16">

**可能原因**

点云数据过少，未检测到托盘腿

**处理方法**

可能存在原因<br />1、相机或叉臂抬升高度发生变化<br />2、腿部被遮挡<br />3、相机或叉臂发生较大俯仰<br />4、腿部成像不足<br />5、反光柱靠托盘腿过近影响相机成像<br />建议通过 palletpro 软件确认后进行配置参数调整、调整反光柱，或调整叉车姿态

</Accordion>

<Accordion title="托盘识别结果返回failed-3或-13">

**可能原因**

托盘腿尺寸为满足需求或未检测到托盘腿

**处理方法**

托盘腿部特特征超过识别范围（可调整腿宽或者托盘宽度范围）

</Accordion>

<Accordion title="托盘识别结果返回failed-4或-14">

**可能原因**

相机安装高度未设置

**处理方法**

设置安装高度参数或进行外参标定

</Accordion>

<Accordion title="托盘识别结果返回failed-5或-15">

**可能原因**

托盘校验未通过；托盘横梁检测异常或托盘倾角过大

**处理方法**

横梁未识别到，建议通过上位机关闭去飞行像素，或调整对接距离使相机更接近托盘，或更改横杆检测比例

</Accordion>

<Accordion title="托盘识别结果返回failed-7">

**可能原因**

视野中托盘腿宽度差异过大

**处理方法**

检查托盘附件是否有反光柱或者靠近墙面影响边缘腿成像

</Accordion>

<Accordion title="托盘识别结果返回failed-8">

**可能原因**

视野中托盘航向角倾斜过大（＞30°）

**处理方法**

摆正托盘检测

</Accordion>

<Accordion title="通信或IO进行避障配置切换，上位机软件上避障配置未变化">

**处理方法**

1、确认备切换的避障配置是否提前设置完成<br />2、确认是否停流启流后再观察是否变化

</Accordion>

<Accordion title="软件显示避障输出结果异常（例：1000）">

**可能原因**

算法版本不匹配

**处理方法**

升级算法固件

</Accordion>

<Accordion title="IO无输出">

**可能原因**

1、S2/S2max确认是否为上拉接法<br />2、S10/S10pro确认固件版本

**处理方法**

1、S2/S2max根据客户情况接入上拉电阻<br />2、S10/S10pro升级到V1.1.050E06_250527后版本的算法固件

</Accordion>

<Accordion title="前方空间无障碍物，避障触发障碍物报警">

**处理方法**

1、检查上位机软件版本<br />2、检查相机固件版本<br />3、检查避障参数配置是否下发

</Accordion>

<Accordion title="库位检测、体积检测、人体检测更新固件后应用算法软件无法打开相机">

**处理方法**

再次重启相机，应用算法软件内全局设置内打开实时刷新

</Accordion>

</AccordionGroup>

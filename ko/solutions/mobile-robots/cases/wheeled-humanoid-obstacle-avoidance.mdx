바퀴형 휴머노이드 로봇은 유연성과 안정성을 바탕으로 특정 환경에서 점차 장점을 보이고 있습니다. 복잡한 작업 환경에서 로봇이 효율적이고 안전하게 이동하려면 장애물을 정확히 감지해야 합니다. Weimeier가 출시한 "Weiba" 바퀴형 로봇은 Meyer Microvision의 RGB-D ToF 카메라 S3를 로봇 하단에 장착해 주변의 다양한 장애물을 감지합니다. 이 구성으로 Weibo 로봇은 낮거나 주변에 있는 장애물을 실시간으로 인식하고 피해 좁고 동적으로 변화하는 산업 환경에서도 안전하게 주행합니다.

Meyer Micro Vision S3 카메라는 RGB(컬러)와 깊이 데이터를 모두 수집합니다. 물체와의 거리를 측정할 뿐 아니라 색상과 텍스처 정보도 포착합니다. "Weibao" 바퀴형 로봇에 풍부한 환경 데이터를 제공해 낮은 장애물을 더 효과적으로 식별하고 피하도록 합니다. 특히 공장 작업장과 물류 창고처럼 복잡한 환경에서 S3 카메라는 로봇의 환경 인식 능력과 장애물 회피 정확도를 크게 높입니다.

S3 카메라는 로봇 하단에 설치하도록 설계되어 로봇이 바닥 가까이에서 이동할 때 낮은 위치의 물체를 감지할 수 있습니다. 이러한 물체는 기존 2D LiDAR나 높은 위치의 카메라로 포착하기 어려운 경우가 많습니다. S3 카메라의 정밀한 깊이 인식 기능을 활용하면 "Weibao" 바퀴형 로봇이 낮은 장애물을 정확히 파악하고 빠르게 대응해 충돌을 피할 수 있습니다.

"Weibao" 바퀴형 로봇 개발팀은 S3 카메라의 높은 프레임 속도와 빠른 응답 성능을 주요 장점으로 꼽았습니다. 빠르게 이동하는 중에도 S3 카메라는 선명하고 정확한 환경 이미지를 포착해 로봇이 작업할 때 높은 유연성과 안전성을 유지합니다. 생산 라인 옆의 자재 운송이나 창고 내 상품 이동처럼 시간과 효율이 특히 중요한 환경에서는 Weibo 로봇이 S3 카메라의 실시간 환경 데이터를 바탕으로 신속하게 장애물 회피 결정을 내려 작업 효율을 크게 높입니다.

S3 카메라는 복잡한 조명 조건에서도 우수한 성능을 유지합니다. 밝은 작업 구역부터 조명이 어두운 모서리까지 안정적인 환경 인식 기능을 제공해 "Weiba" 바퀴형 로봇이 다양한 환경에서 원활하게 작업하도록 합니다.

## 배경

과제: AGV를 도입한 대부분의 창고에서도 물품은 여전히 수작업으로 적재됩니다. 이 때문에 WMS 시스템이 각 창고 위치의 실제 점유 상태를 실시간으로 판단하지 못하고 AGV에 부정확한 정보를 제공할 수 있습니다. 점유된 위치를 제때 파악하지 못하면 AGV가 잘못된 지시를 받아 작업 효율이 떨어질 뿐 아니라 다른 화물이나 장비와 충돌하기 쉽습니다. 심한 경우 안전사고가 발생해 사람과 장비의 안전을 위협할 수 있습니다.

이러한 문제를 해결하기 위해 창고 위치 상태 인식 시스템이 등장했습니다. 자동 측위, 실시간 데이터 동기화, 정밀한 재고 관리 기능으로 창고 리소스 배분을 최적화하고 인적 오류를 줄여 운영 효율을 높입니다.

**창고 위치 상태 인식 센서의 일반적인 문제 두 가지:**

- 단일 포인트 LiDAR가 전송하는 창고 위치 정보는 정확도가 부족해 적재 사고를 일으키기 쉽습니다.

일부 창고에서는 창고 위치 감지에 단일 포인트 LiDAR를 사용합니다. 이 거리 측정 레이더는 한 번에 레이저 빔 하나만 방출해 물체 표면에 점을 만듭니다. 상자나 팔레트 사이의 틈을 놓쳐 창고 위치 상태를 잘못 인식하고 적재 사고를 일으킬 수 있습니다.

- RGB 카메라만 사용하는 방식은 보정 방법이 제한적이며 오판과 정보 누락 위험이 있습니다.

RGB 카메라로 위치 상태를 판단할 때는 다음과 같은 단점이 있습니다.

- 딥 러닝으로 대상을 감지할 때 훈련 세트에 없는 물체가 창고에 들어오면 오탐이 발생해 잘못된 정보를 제공할 수 있습니다.

- 화물 높이 정보가 없어 적재 작업을 계획하기 어렵습니다.

- 초광각 어안 카메라는 창고 위치를 판단할 때 가장자리 왜곡이 발생합니다. 모델 훈련과 예측의 정확도를 낮추고 서버 비용을 높입니다.

## 솔루션 개요

Meyer Micro Vision은 **RGB-D 카메라**로 창고 위치의 3차원 데이터와 색상 정보를 제공하고, 화물의 유무, 배치 규격, 비정상 점유를 자동으로 식별합니다. 3D 비전 지능형 AI 시스템으로 화물 종류를 구분해 배차 시스템으로 출력하며 지능형 입출고 검사를 구현합니다. 카메라에 연산 기능이 내장되어 외부 산업용 컴퓨터가 필요하지 않습니다. Meyer Micro Vision은 창고 상태 인식을 위한 통합 소프트웨어·하드웨어 솔루션과 전문적인 모델 훈련, 배포 안내 등의 지원도 제공합니다.

## 핵심 장점

- 종합 모니터링: 3차원 데이터와 색상 정보를 수집하고 AI 기술과 결합해 창고 위치 상태를 정확히 식별합니다.

- 간편한 배포: 창고 위치 상태 인식 알고리즘이 카메라에서 실행되어 배포 및 유지관리 비용을 줄입니다.

- 유연한 통신: TCP/IP, UDP, HTTP 등 다양한 통신 방식을 지원하며 JSON 형식으로 제어 시스템에 데이터를 실시간 보고합니다.

- 효율 향상: 창고 위치 정보를 실시간으로 전송해 배차 시스템이 작업을 빠르고 정확하게 할당하도록 지원합니다.

## 작업 설명

> **카메라 선택 권장 사항: S10Pro**

> **카메라 설치 방법**

고정 설치: 카메라를 기둥이나 천장에 설치할 때는 모니터링할 창고 위치 바로 위에 설치하는 것이 좋습니다.

> **구체적인 워크플로**

Step1: 상위 제어 시스템이 작업 지시를 전송하면 3D 비전 카메라가 이미지를 수집하고 지정된 창고 위치 상태를 정확히 확인합니다.

Step 2: 창고 위치 감지 결과를 JSON으로 상위 시스템에 보고해 배차 시스템의 지능형 입출고 관리를 지원합니다.

통신 방식: TCP, UDP, HTTP

> **배포 참고 사항**

적용 환경: 먼지가 심하거나 결로가 발생하는 열악한 환경은 피해야 합니다.

설치 요구 사항: 카메라 장착 브래킷의 강성과 안정성을 확보하세요.

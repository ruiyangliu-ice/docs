## 프로젝트 배경

<Frame>
  <img src="/images/external/c60631c163bf9220037d5aed.avif" alt="창고 전경" />
</Frame>

중국 남부의 한 자동화 창고는 화물 운반 공정을 최적화하기 위해 여러 대의 AGV를 도입했습니다. 그러나 일상적인 창고 운영 중에는 수작업으로 화물을 운반하는 상황이 불가피하게 발생해 시스템이 파악한 창고 위치 점유 상태 정보가 제때 업데이트되지 않았습니다. 정보가 지연되면 AGV가 잘못된 작업 지시를 받아 운영 효율이 떨어질 뿐 아니라 안전 문제까지 발생할 수 있습니다.

이를 해결하기 위해 프로젝트는 모든 위치를 모니터링하고 현재 상태 정보를 WMS에 실시간으로 보내는 빈 위치 상태 감지 시스템을 사용합니다. **빈 위치의 디지털 관리**를 구현하고 AGV가 화물을 더 효율적이고 안정적이며 안전하게 입출고하도록 지원합니다.

## 솔루션

### 창고 위치 상태 인식 센서 선택

**• 단일 포인트 LiDAR:** 단일 포인트 LiDAR는 한 번에 레이저 빔 하나만 방출해 물체 표면에 점을 만듭니다. 이 거리 측정 방식은 상자나 팔레트 사이의 틈을 놓치기 쉬워 현재 창고 위치에 화물이 없거나 비어 있다고 잘못 판단하고 적재 사고를 일으킬 수 있습니다.

**• RGB 카메라:** 딥 러닝으로 대상 영역을 감지할 때 훈련 세트에 없는 물체가 창고에 들어오면 오탐이 발생해 잘못된 창고 위치 판단 정보를 제공할 수 있습니다. 또한 화물 적재 요구 사항이 있는 창고 위치를 판단할 때 RGB 카메라가 제공하는 정보에는 화물 높이가 없어 적재 작업을 계획하기 어렵습니다.

**• Meier Micro Vision RGB-D 카메라:** 창고 위치의 3차원 데이터와 색상 정보를 제공하고 연산 기능이 내장되어 외부 산업용 컴퓨터가 필요하지 않습니다.

Myer Micro Vision은 창고 위치 상태 인식 소프트웨어와 하드웨어의 통합 솔루션뿐 아니라 전문적인 모델 훈련과 배포 안내 등의 서비스도 제공합니다. 전체 솔루션의 비용 효율이 높고 배포가 간편해 창고 전역에 센서 수백 대를 설치하고 창고 위치 상태를 종합적으로 실시간 모니터링했습니다.

### 솔루션 상세 설명

Meyer Micro Vision이 RGB-D 카메라를 기반으로 개발한 3D 비전 창고 위치 상태 인식 솔루션은 창고 위치의 3D 포인트 클라우드 데이터와 RGB 데이터를 수집해 정확히 식별합니다. 각 창고 위치의 점유 상태와 화물 높이 정보를 실시간으로 모니터링하고 모바일 로봇에 상세 데이터를 제공합니다. 창고 위치를 디지털로 관리하고 정확한 입출고 작업을 보장합니다. 창고 위치 인식 알고리즘이 카메라에 내장되어 별도의 외부 연산 장치가 필요하지 않으므로 배포 복잡성과 프로젝트 비용이 줄어듭니다.

**• 종합 모니터링:** 3차원 데이터와 색상 정보를 수집하고 AI 기술과 결합해 창고 위치 상태를 정확히 식별합니다.

**• 간편한 배포:** 창고 위치 상태 인식 알고리즘이 카메라에서 실행되어 배포와 유지관리 비용이 줄어듭니다.

**• 유연한 통신:** TCP/IP, UDP, HTTP 등 다양한 통신 방식을 지원하며 JSON 형식으로 제어 시스템에 데이터를 실시간 보고합니다.

**• 효율 향상:** 창고 위치 정보를 실시간으로 전송해 배차 시스템이 작업을 빠르고 정확하게 할당하도록 지원합니다.

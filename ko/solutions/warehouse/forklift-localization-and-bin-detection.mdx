기존 창고·물류 관리에서 수동 지게차와 창고 위치 관리는 핵심 요소지만 효율적인 운영에는 여러 제약이 따릅니다.

**• 최적화되지 않은 경로와 낮은 효율**: 수동 지게차의 주행 경로가 운전자의 경험에 좌우되어 불필요한 주행 시간이 늘고 작업 강도가 높아집니다.

**• 위치 정보 지연과 배차 혼선**: 실시간 위치 상태 업데이트가 없어 작업 할당 오류가 발생하고 전체 운영 효율이 떨어지며 안전 문제까지 생길 수 있습니다.

**• 관리하기 어려운 안전 위험**: 좁은 통로와 높은 선반 환경에서는 지게차 충돌 사고가 자주 발생해 작업자와 장비의 안전을 위협합니다.

이러한 문제를 해결하기 위해 일부 프로젝트에서는 UWB 기술 기반 측위 솔루션으로 지게차 경로를 최적화하고 안전성을 높이려 했습니다. 그러나 UWB 측위 시스템은 실제 적용에서 여러 한계를 보입니다.

**1. 신호 간섭에 취약**: 금속 선반이나 물품이 복잡하게 적재된 환경에서는 UWB 신호가 간섭받기 쉬워 측위 데이터가 불안정합니다.

**2. 제한적인 정확도와 단일 기능**: UWB 측위 정확도는 20-50 cm에 불과합니다. 위치 데이터만 제공하고 각도 정보는 지원하지 않아 창고 환경의 정밀 관리 요구를 충족하기 어렵습니다.

**3. 배포와 비용 부담**: 대형 창고에서는 UWB 기지국과 복잡한 배선을 다수 설치해야 하며 배포 비용이 수십만 위안에 달합니다.

**4. 화물 추적의 한계**: UWB만으로 화물 위치를 직접 추적하기 어렵습니다. 일반적으로 RFID 등 다른 기술을 함께 사용해야 하므로 시스템 복잡성과 비용이 더 증가합니다.

## 솔루션

Meyer Micro Vision은 오랫동안 물류·창고 자동화 분야에 집중해 업계의 문제를 심층적으로 연구하고 여러 선도 기업과 전략적 협력 관계를 구축했습니다. 지금까지 3D 비전 센서를 수만 대 공급하며 업계를 선도하고 있습니다. Meyer Micro Vision은 3D 비전 센서를 기반으로 **RTLS 수동 지게차 측위 시스템**과 **창고 상태 인식 시스템**을 출시해 창고와 물류를 위한 통합 지능형 솔루션을 제공합니다.

> **수동 지게차 측위 시스템**

수동 지게차 측위 시스템은 지게차 상단에 3D 비전 카메라를 설치해 천장과 벽의 자연 특징을 포착합니다. UWB의 20-50 cm 정확도보다 크게 향상된 1-5 cm의 정밀 측위를 구현합니다. 또한 **IMU 보조 측위**와 결합해 가림이 있거나 시각 정보가 부족한 환경에서도 지속적이고 안정적인 측위를 보장합니다.

이 시스템은 실시간 측위뿐 아니라 운전자에게 주변의 사람, 차량, 잠재적인 위험을 실시간으로 알리고 자동 진입 금지 구역을 설정합니다. 지게차 충돌과 기타 안전사고 발생 가능성을 줄여 작업자와 장비에 더 안전한 환경을 제공합니다.

Meyer Micro Vision의 **비전 RTLS 기술** 기반 솔루션은 정밀 측위와 유연한 하드웨어 지원을 통해 Bluetooth, UWB 같은 기존 RTLS 솔루션보다 높은 정확도, 낮은 설치 비용, 빠른 배포 능력을 제공합니다. 수동 지게차의 지능형 업그레이드에 적합합니다. 측위 시스템은 창고 상태 인식 시스템과 연동해 화물 배치를 동적으로 추적하고 대상 창고 위치 상태를 실시간으로 업데이트합니다. 불필요한 작업을 방지하고 작업 할당과 운행 경로를 유연하게 조정할 수 있습니다.

> **재고 위치 상태 인식 시스템**

Meyer Micro Vision은 **RGB-D 카메라**로 창고 위치의 3차원 데이터와 색상 정보를 제공하고 화물의 유무, 배치 규격, 비정상 점유를 자동으로 식별합니다. 이 솔루션은 3D 비전 지능형 AI 시스템으로 화물 종류를 구분하고 데이터를 배차 시스템으로 출력해 지능형 창고 입출고 작업을 완료합니다.

**핵심 기능**

**• 종합 모니터링**: 창고 위치의 3차원 데이터와 색상 정보를 수집하고 AI 기술과 결합해 창고 위치 상태를 정확히 식별합니다.

**• 간편한 배포**: 알고리즘이 카메라에서 실행되므로 외부 산업용 컴퓨터가 필요하지 않아 배포와 유지관리 비용이 줄어듭니다.

**• 유연한 통신**: TCP/IP, UDP, HTTP 등 다양한 통신 방식을 지원하며 JSON 형식으로 제어 시스템에 데이터를 실시간 보고합니다.

**• 효율적인 배차**: 위치 상태 정보를 실시간으로 전송해 배차 시스템이 작업을 빠르게 할당하고 운영 효율을 높이도록 합니다.

## RTLS와 위치 인식의 연동

Myer Micro Vision 솔루션은 RTLS 측위 시스템과 창고 상태 인식 시스템을 긴밀하게 연동해 지게차, 화물, 창고 위치를 통합 관리하고 작업 배차의 정확도와 효율을 높입니다.

- 측위와 인식 통합: 시스템이 지게차 위치와 창고 상태를 실시간으로 동기화해 화물을 동적으로 추적하고 작업 중 화물을 안전하고 정확하게 관리합니다. 3차원 데이터와 위치 정보를 원활하게 통합해 창고 상태 가시성을 크게 높입니다.

- 작업 경로 동적 조정: 실시간으로 업데이트되는 화물 및 창고 위치 정보를 바탕으로 지게차 작업 경로를 유연하게 조정해 위치 충돌과 오작동을 방지합니다. 지게차의 빈 차 이동 시간을 줄이고 전체 운영 효율을 높여 운영 비용을 절감합니다.

- 리소스 배분 최적화: 정확한 데이터 분석으로 창고 위치와 장비의 활용도를 종합적으로 최적화해 창고 공간 활용 효율을 높입니다.

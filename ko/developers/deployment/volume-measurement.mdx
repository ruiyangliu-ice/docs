## 1. 소개

부피 감지 알고리즘은 Meyer Micro Vision(MRDVS)에서 개발했으며 특히 부피 측정 시나리오에 사용됩니다. 물류, 속달 배송, 석탄 야적장, 창고 및 기타 시나리오에서 자체 개발한 M 시리즈 카메라에 사용됩니다. 볼륨 감지 알고리즘은 카메라 내부에서 실행되므로 외부 산업용 컴퓨터가 필요하지 않습니다. 알고리즘은 정적 및 동적 탐지를 지원합니다.

소프트웨어는 사용자가 카메라의 외부 매개변수를 교정하고 감지 범위를 그리기만 하면 단일/다중 물체를 감지하고 측정할 수 있습니다. 외부 매개변수 교정은 자동으로 교정 결과를 얻고 다중 카메라 접합 감지를 지원할 수 있습니다.

이 기사에서는 주로 Meyer 마이크로비전 볼륨 감지 기능의 배포 및 사용을 소개합니다.

## 2. 볼륨 감지 애플리케이션 시나리오

### 2.1 물류산업

1. 물품의 부피, 중량 등의 정보를 획득함으로써 정밀한 작업을 달성할 수 있으며, 이를 통해 차량 적재율을 향상시키고 차량 비용, 운송 비용 및 인건비를 절감하며 전체 프로세스에 걸쳐 비용 절감 및 효율성 향상을 달성할 수 있습니다.

2. 원클릭으로 상품/박스의 부피와 무게를 빠르게 감지하고 화물을 계산할 수 있습니다. 바닥형 저울과 함께 트럭 및 컨테이너 트럭을 효과적으로 측정하여 전체 차량이 과중량이 아닌지 확인할 수도 있습니다.

### 2.2 특송산업

신속 배송을 보내고 받을 때 지능형 부피 측정 장비의 도움으로 상품의 부피 및 무게와 같은 정보를 신속하게 얻을 수 있습니다. 데이터는 내부 시스템과 연결되어 있어 검색 및 검토가 용이하여 데이터 손실 문제를 피할 수 있을 뿐만 아니라 수동 투자를 효과적으로 줄이고 업무 효율성을 향상시킵니다.

### 2.3 석탄 채굴 산업

대형 석탄광석 더미의 부피를 측정하는 전통적인 방법은 일반적으로 토탈 스테이션과 RTK 단일 지점 측정 방법을 사용하며 특징점을 정확하게 수동으로 추출해야 합니다. 그러나 대형 석탄광석 더미는 일반적으로 오르기가 불편하기 때문에 정확한 특징점을 추출하는 것이 극히 어렵습니다. 동시에 단일 지점 측정 방법은 데이터 밀도가 낮고 석탄 광석 더미의 부피를 정확하게 얻을 수 없어 작업 효율성이 매우 낮습니다.

### 2.4 창고 산업

물류, 창고업, 심지어 일반 산업 분야에서도 AGV는 물품을 운반하며, 창고 입출고 시 물품의 부피와 무게를 확인해야 합니다.

### 3. 문제를 해결하세요

물류, 창고업 등 산업 산업에서 품목의 수량과 수량을 파악하고 품목의 입출고를 마스터하는 것은 생산 재고 관리에 큰 의미를 갖습니다. 그러나 기존 솔루션은 기술적 수단 부족, 측정 오류가 크다, 수동 관리 효율성이 낮고 감독이 어렵다는 등의 문제에 항상 직면해 있습니다.

Meyer Micro Vision ToF 카메라 볼륨 감지는 레이더, 구조광 및 2D 카메라보다 더 많은 장점을 가지고 있습니다. Meyer Micro Vision 카메라는 배포가 간단하며 카메라 펌웨어 브래킷, 전원 코드 및 네트워크 케이블만 필요합니다. 알고리즘은 중복된 산업용 컴퓨터 구성 없이도 카메라 내부에서 실행됩니다. 레이더에 비해 사용이 편리하고 비용도 저렴합니다.

### 7.3 용량 감지 외부 ​​매개변수 교정

1. 카메라 IP 구성이 완료되고 방화벽이 닫힌 후 배포 2단계를 진행합니다. 소프트웨어를 열어 알고리즘의 외부 매개변수를 교정합니다. 특정 기준 접지와 카메라 설치 각도 편차를 보정해야 합니다.

2. 카메라 구성을 클릭하고 매개변수 보정 인터페이스로 전환합니다. 외부 매개변수 보정을 수행하기 전에 x, y, z, 롤, 피치, 요가 0인지 확인하십시오. 0이 아닌 경우 모든 매개변수를 0으로 되돌리고 [**저장 및 보내기**] 버튼을 클릭해야 합니다. 매개변수가 성공적으로 저장되면 외부 매개변수를 다시 교정하십시오.

### 8.3 HTTP 통신

1. 볼륨 감지 결과는 HTTP 통신, 포트 번호: 14952를 통해 얻을 수 있습니다.

2. 상위 계층 시스템(WMS)은 비주얼 전송 결과로부터 신호를 획득하고 비주얼은 JSON 형식의 문자열을 반환합니다.

| WMS -> 비전 |﻿﻿﻿﻿
| --- | --- |
| 결과 | 의미 |
| 결과_업데이트 | 시각적 결과 반환 신호 트리거 |
| 비전 -> WMS |﻿﻿﻿﻿
| 결과 | 의미 |
| IP | 데이터 소스 카메라 IP |
| TM | 볼륨 감지 상태 |
| 각도 | 선택한 객체의 각도 |
| CX | 프레임이 선택된 객체의 x 좌표 |
| 사이 | 프레임이 선택된 객체의 y 좌표 |
| cz | 선택된 객체 z 좌표 프레임 |
| 신장 | 물체 높이 |
| 길이 | 물체 길이 |
| 폭 | 개체 너비 |
| 유효한 | 신뢰도가 너무 낮으면 false, 그렇지 않으면 true |
| TM_DATA_TIME | 볼륨 감지 데이터 추출 시간 |
| TM_RES_TIME | 볼륨 감지 결과 출력 시간 |
| 별명 | 카메라에서 구성 가능한 볼륨 감지 카메라 이름 |
| ret_code | 반환 코드, 0은 성공 |

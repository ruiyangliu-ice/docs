## 1. 소개

### 1.1 목적 및 범위

이 문서는 제품을 통합하고 사용하는 개발자를 위해 S10 카메라의 사양과 주요 설계 정보를 설명합니다.

### 1.2 용어

| RGBD 카메라 | 시간 및 공간 정렬된 RGB 이미지와 깊이 이미지를 출력하는 카메라 |
| --- | --- |
| 깊이 | 컬러 또는 흑백 스트림과 유사한 비디오 스트림으로, 각 픽셀이 카메라로부터의 거리를 나타냄 |
| FOV | 시야각: 카메라가 캡처하는 장면의 각도 범위로, 수평(HFOV), 수직(VFOV), 대각선(DFOV) 시야각을 포함 |
| IR 카메라 | 적외선 카메라로, 이 제품에서는 주로 근적외선 대역을 캡처 |
| RGB 카메라 | 컬러 카메라 |
| 깊이 카메라 | 일반적으로 도트 프로젝터, IR 카메라, 깊이 프로세서로 구성된 깊이 모듈이 있는 카메라. 컬러 모듈은 일반적으로 RGB 카메라임 |
| SoC | 이미지 처리, 깊이 계산, 응용 알고리즘에 사용하는 시스템 온 칩 |
| DTOF | 비행 시간 측정 기반의 거리 감지 기술 |

### 1.3 작동 원리

이 제품은 긴 측정 거리, 우수한 정확도, 낮은 비용, 강한 간섭 내성, 안정적인 데이터를 제공하는 DTOF 3D 깊이 카메라입니다. 통합 RGB 모듈이 컬러 이미지를 캡처하며, 카메라 내부에서 RGB 이미지와 깊이 이미지를 정렬합니다.

## 2. 제품 개요

이 소형 산업용 RGBD 카메라는 0.3m~8m 범위의 깊이 데이터를 제공합니다. RGB 이미지와 깊이 이미지를 공간 및 시간 기준으로 정렬합니다. 통합 SoC는 깊이, 강도, RGB, 포인트 클라우드, 장애물 감지 결과를 출력합니다. 이더넷 외에도 2개의 IO 입력과 2개의 IO 출력을 통해 4개 장애물 회피 구역을 전환하고 비상 정지, 감속, 안전 상태 신호를 출력합니다.

## 6. 빠른 시작

### 6.1 연결 테스트

<Frame>
  <img src="/images/legacy/cb764c04-73c2-4cdd-9381-98fe1c201737/24cb2351-298f-4381-8170-c34cf1444269.avif" alt="연결 테스트" />
</Frame>

1.  카메라를 24V/2A DC 전원 공급 장치에 연결합니다. 파란색 전원 표시등이 천천히 깜박이면 전원이 정상적으로 켜진 것입니다.

2.  Category 6 이더넷 케이블로 카메라를 컴퓨터에 연결합니다.

3.  카메라의 공장 출하 기본 IP 주소는 192.168.100.82입니다.

4.  컴퓨터 IP를 카메라와 같은 서브넷으로 설정하고 컴퓨터 방화벽을 비활성화합니다.

5.  호스트 소프트웨어를 실행하여 카메라 이미지를 확인합니다.

6.  호스트 소프트웨어가 필요하면 MRDVS 영업 또는 기술 지원에 문의하세요.

### 6.2 IO 배선

| 인터페이스 | 유형 | 참고 |
| --- | --- | --- |
| IN1 | 입력 | 액티브 하이 디지털 입력, 24V/오픈 입력 지원 |
| IN2 | 입력 | 액티브 하이 디지털 입력, 24V/오픈 입력 지원 |
| OUT1 | 출력 | 접지/오픈 디지털 출력, 24V 풀업 지원 |
| OUT2 | 출력 | 접지/오픈 디지털 출력, 24V 풀업 지원 |
| COM | GND | 디지털 출력 공통 접지, IN1은 IN_COM1과 함께 사용 |

## 8. 문제 해결

| 번호 | 문제 | 해결 방법 |
| --- | --- | --- |
| 1 | 카메라를 열 때 응답하지 않음 | 방화벽이 비활성화되어 있는지 확인합니다. |
| 2 | 호스트 소프트웨어에서 여러 IP 주소를 감지함 | LAN의 다른 장치가 같은 IP를 사용하고 있습니다. 필요한 IP 주소를 선택합니다. |
| 3 | 카메라를 연 후 데이터가 불안정함 | 기가비트 이더넷 케이블을 사용합니다. Fast Ethernet에서는 데이터가 처음에 불안정할 수 있습니다. |
| 4 | 소프트웨어 설치 위치 | C 드라이브의 기본 위치에 설치하면 권한 설정의 영향을 받을 수 있습니다. |

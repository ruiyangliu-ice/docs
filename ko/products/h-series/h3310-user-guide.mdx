## 1. 소개

### 1.1 목적 및 범위

이 문서는 제품을 통합하고 사용하는 개발자를 위해 H3310 카메라의 사양과 설계 세부 정보를 제공합니다.

### 1.2 용어

| 베이스라인 | 스테레오 깊이 카메라에서 베이스라인은 두 카메라 사이의 수평 거리입니다. 더 정확히는 두 광축 사이의 거리이며 일반적으로 밀리미터(mm) 단위로 측정합니다. |
| --- | --- |
| 깊이 | 컬러 또는 흑백 비디오 스트림과 마찬가지로 깊이 스트림은 각 픽셀에 값을 할당합니다. 이 값은 카메라에서 해당 픽셀까지의 거리를 나타냅니다. |
| FOV | 시야각입니다. 카메라가 캡처할 수 있는 장면의 각도 범위로, 수평 FOV(HFOV), 수직 FOV(VFOV), 대각선 FOV(DFOV)를 포함합니다. |
| IR 카메라 | 적외선 카메라입니다. 이 제품에서는 주로 근적외선 스펙트럼의 이미지를 캡처합니다. |
| RGB 카메라 | 컬러 카메라입니다. |
| 깊이 카메라 | 일반적으로 도트 프로젝터, IR 카메라, 깊이 프로세서로 구성된 깊이 이미징 모듈을 갖춘 카메라입니다. 컬러 이미징 모듈은 일반적으로 RGB 카메라입니다. |
| SoC | 이미지 데이터를 처리하고 깊이를 계산하며 응용 알고리즘을 실행하는 시스템 온 칩입니다. |

### 1.3 구조광 기술

쌍안 구조광은 스테레오 비전과 구조광 투영을 결합한 일반적인 3D 비전 측정 기술입니다. 두 대의 카메라와 구조광 프로젝터를 사용하여 물체의 3D 형상과 깊이를 캡처합니다.

작동 과정은 다음과 같습니다.

1.  구조광 투영: 프로젝터가 대상 물체에 구조광을 투사합니다. 일반적으로 적외선 또는 가시광선 광원이 빛을 생성하며, 격자 또는 스펙클 패턴을 사용해 특수 인코딩하거나 변조합니다. 투사된 빛은 물체 표면에 밝거나 어둡거나 왜곡된 패턴을 만듭니다.

2.  스테레오 이미징: 구조광이 투사되는 동안 두 카메라가 대상 물체를 동시에 캡처합니다. 카메라 간격은 사람의 두 눈과 같이 일정하게 고정되어 있습니다. 두 카메라는 투사된 패턴과 물체의 외형을 함께 기록합니다.

3.  이미지 캡처: 카메라가 왼쪽 및 오른쪽 이미지를 캡처합니다. 이 이미지에서 투사 패턴의 왜곡이나 변위가 물체의 깊이 정보를 인코딩합니다.

4.  이미지 처리: 왼쪽과 오른쪽 이미지의 차이를 사용해 표면 깊이를 계산합니다. 이 차이를 투사 패턴과 결합하여 물체의 3D 포인트 클라우드 모델을 생성합니다.

5.  측정 및 재구성: 생성된 3D 포인트 클라우드를 사용해 물체의 형상과 치수를 측정, 분석 또는 재구성할 수 있습니다.

## 2. 제품 개요

H3310은 무게가 400g인 RGB-D 산업용 구조광 카메라입니다. 복합 로봇이나 소형 로봇 팔에 장착할 수 있습니다. 30-60cm의 인식 거리에서 고정밀 3D 데이터와 고해상도 RGB 데이터를 제공합니다. 통합 컴퓨팅 플랫폼이 AI 응용 알고리즘을 독립적으로 실행하여 산업용 로봇의 인식, 검사 및 위치 결정 작업을 수행합니다.

### 2.2 특징

- 통합 컴퓨팅 플랫폼: 6.0 TOPS의 컴퓨팅 성능을 제공하며 외부 CPU 없이 응용 알고리즘을 실행합니다.

- 3D 데이터 출력: 구조광 이미징과 AI 알고리즘을 결합해 고정밀 3D 데이터를 제공합니다.

- 컴팩트한 설계: 무게는 400g이며 배포를 위한 개방형 인터페이스를 제공합니다.

- 대량 생산: 안정적인 대량 생산을 지원합니다.

## 7. FAQ

| 번호 | 문제 | 해결 방법 |
| --- | --- | --- |
| 1 | 카메라를 열려고 해도 응답하지 않음 | 방화벽이 비활성화되어 있는지 확인하세요. 방화벽을 반드시 비활성화해야 합니다. |
| 2 | 호스트가 여러 IP 주소를 감지함 | 카메라가 LAN에서 이미 사용 중인 IP 주소를 할당받았습니다. 필요한 IP 주소를 선택하여 카메라를 여세요. |

| 4 | 소프트웨어 설치 위치 | 기본 C 드라이브에 소프트웨어를 설치하려면 권한 설정이 필요할 수 있습니다. |

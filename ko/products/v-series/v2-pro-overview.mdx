## 1. 제품 개요

### 1.1 소개

MRDVS V2 Pro는 상향 시각 SLAM 카메라, 2D LiDAR 및 IMU의 데이터를 융합하는 내비게이션 컨트롤러입니다. 통합 컴퓨팅 플랫폼에는 외부 산업용 컴퓨터가 필요하지 않습니다. 로봇이 내비게이션 영역을 이동할 때 카메라는 환경 데이터를 캡처하고 모바일 로봇 위치 파악 및 내비게이션을 위한 지도를 생성합니다.

### 1.2 기능

#### 1.2.1 모바일 로봇 네비게이션

- 108°×57° 상향 카메라, 270° 스캔 범위의 2D LiDAR, ±1cm 반복 위치 지정 정확도를 위한 IMU의 데이터를 융합합니다.

- 6 TOPS 프로세서와 매핑을 위한 시각적 인터페이스를 사용합니다.

#### 1.2.2 수동 지게차 개조

- 수동 지게차 위치 파악 데이터 전송을 위한 Wi-Fi 6가 포함되어 있습니다.

- 음성 안내 및 경고를 위한 3핀 오디오 출력을 제공합니다.

### 1.3 애플리케이션

1.3.1 공장 물류: 창고, 하역장 및 작업장 간 AGV 및 자율 지게차를 통한 원활한 실내외 운영을 지원합니다.

1.3.2 수동 지게차 업그레이드: UWB를 더 높은 정확도(3cm 미만), 더 낮은 배포 비용 및 더 짧은 배포 시간으로 대체합니다.

#### 3.3.2 Wi-Fi 통신 모듈

V2 Pro 내비게이션 컨트롤러에는 안정적인 고속 Wi-Fi를 위한 무선 네트워크 어댑터가 포함되어 있습니다. 다중 사용자 MIMO(MU-MIMO)는 여러 장치가 동일한 대역을 공유할 때 채널 용량을 늘리고 듀얼 스트림 작동은 최대 1200Mbps의 속도를 지원합니다. 어댑터는 2.400GHz~2.4835GHz ISM 대역에서 IEEE 802.11ax/ac/a/b/g/n을 지원합니다.

| 데이터 요금 | Wi-Fi:<br />• 802.11b: 11Mbps<br />• 802.11a/g: 54Mbps<br />• 802.11n: MCS0~15<br />• 802.11 AC: MCS0~9<br />• 802.11ax: HE0~11<br />블루투스: 1Mbps 또는 2Mbps, 최대 3Mbps |

#### 5.2 연결 테스트

<Frame>
  <img src="/images/legacy/dfd4c3be-816c-4ffc-81ac-13141e3c2644/9d8858e6-85a0-492f-a353-4e5adf1f81a2.avif" alt="연결 테스트" />
</Frame>

#### 5.2.1 카메라 전원 켜기

- DC 케이블의 한쪽 끝을 카메라에 연결하고 다른 쪽 끝을 24V/3A DC 어댑터에 연결합니다.

- 파란색 전원 표시등이 천천히 깜박이면 전원이 정상적으로 켜졌음을 나타냅니다.

- 표시등이 꺼져 있거나 빨간색으로 켜져 있는 등 비정상적인 경우에는 전원 커넥터와 어댑터를 확인하세요.

#### 5.2.2 네트워크 구성

- 카테고리 6 이더넷 케이블을 사용하여 카메라를 컴퓨터에 직접 연결합니다.

- 공장 출하시 카메라 IP 주소는 192.168.100.201 입니다.

- 컴퓨터 IP를 동일한 서브넷으로 설정하고 방화벽을 비활성화합니다.

- LAN 컴퓨터에서는 웹 브라우저에 192.168.100.201:9998을 입력하면 비밀번호 없이 로그인할 수 있습니다.

#### 5.2.3 소프트웨어 실행

- 호스트 소프트웨어를 실행하고, 카메라 이미지를 획득하고, 이미지가 선명하고 안정적인지 확인합니다.

- 섹션 5.3 또는 LxCameraViewer 사용자 가이드를 참조하세요.

추가 도움이 필요하면 MRDVS 영업 또는 기술 지원에 문의하세요.

#### 5.3 인터페이스 개요

<Frame>
  <img src="/images/legacy/4e6942a8-55ab-4030-9609-4c1392a560fa/69362392-66e2-4bd2-8fc3-72e4a3e1efbe.avif" alt="V2 Pro 인터페이스" />

1. 빨간색 메뉴바 : 매핑, 리로컬라이제이션, 지도 관리 등의 기능을 엽니다.

2. 파란색 상자: 로드된 2D 외곽선 지도와 차량 위치 파악 아이콘이 표시됩니다.

3. 녹색 상자: 위쪽을 향한 카메라의 실시간 RGB 이미지를 표시합니다.

4. 노란색 상자: 차량 좌표, 차량 속도, 커서 좌표와 함께 위치 재설정 신호 강도(녹색: 매우 좋음, 파란색: 좋음, 빨간색: 나쁨)를 실시간으로 표시합니다.

1. 매핑 시작

3. 매핑 인터페이스

<Frame>
  <img src="/images/legacy/2568abac-69b0-4e56-af86-53575c614918/12edb330-0198-49f8-a1de-cbcc05192b4a.avif" alt="매핑 인터페이스" />

- 매핑 시 차량의 직선 속도는 0.5m/s 이하, 각속도는 20도/s 이하로 유지하세요.

- 인터페이스는 레이저 윤곽선과 상향 카메라 이미지를 표시합니다.

4. 대략적인 매핑 경로를 미리 계획하세요. 저속(&lt;0.2m/s)으로 꾸준히 이동하고, 직선과 직각 회전을 사용하고, 반복되는 경로와 무작위 진동을 피하십시오. 같은 방향의 경로를 따라 약 1/3의 시각적 중첩을 유지합니다. 평행 경로는 일반적으로 약 2m 떨어져 있어야 합니다. 아래와 같이 매핑 영역이나 로봇 경로를 최대한 완벽하게 커버하세요.

| 1 | 카메라를 열면 반응하지 않습니다 | 카메라 통신을 차단하지 않도록 방화벽 비활성화 |
| 2 | 호스트 소프트웨어가 여러 IP 주소를 감지 | LAN 주소 충돌을 해결하고 올바른 IP 주소 선택 |
| 3 | 카메라 스트림이 불안정합니다 | 기가비트 이더넷 케이블을 사용하십시오. 고속 이더넷은 초기 불안정을 초래할 수 있음 |
| 4 | 소프트웨어 설치 위치 | 기본 드라이브 C 위치 대신 적절한 읽기 및 쓰기 권한이 있는 디렉터리 사용 |

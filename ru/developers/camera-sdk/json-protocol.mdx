## Интерфейс

- Сервер TCP-сокетов. Камера выступает в роли сервера. Порт по умолчанию — `5501`.

- Содержание сообщения: текстовый формат с поддержкой ASCII. Рекомендуется JSON.

## Запрос и ответ

**1. Робот (ПЛК) —> система технического зрения**

```json
{"action": "3D"}
```

Этот запрос запускает задачу 3D-видения.

**2. Система технического зрения —> робот (ПЛК)**

Камера возвращает результат обнаружения со следующими полями:

| Поле | Требуется | Описание |
| --- | --- | --- |
| результат | Да | Статус обнаружения целого числа. `0` указывает, что локализация не удалась и результат недоступен. Любое другое значение указывает на успех и указывает количество возвращаемых результатов. |
| пос | Да | Значения положения X, Y, Z, RX, RY и RZ для обнаруженного объекта в системе координат робота. Единицы измерения – миллиметры и градусы. `-180° ≤ RZ ≤ 180°`. При сбое обнаружения все значения устанавливаются на `0`. |

**Примеры**

| Обнаружение удалось | `{"result":2,"pos":[100,107.3,1530,179,89,89,112,432.1,1500,179,89,75]}` |
| --- | --- |
| Обнаружение не удалось | `{"result":0,"pos":[0,0,0,0,0,0]}` |

## Получить глобальную информацию

### Обновление основной программы

URL-адрес запроса

`http://<ip>:9998/tool/rob/quiet_update`

Пример запроса:

```json
{

"url":"http://192.168.2.179:8080/hontai_2.1.20211114.zip"

}
```

### Установить высоту камеры

URL-адрес запроса `http://<ip>:9998/tool/rob/setHeightRequest` Пример:

```json
{

"height": 1.3

}
```

Пример ответа:

```json
{

"code":200, "message":"success",

}
```

`http://<ip>:9998/tool/rob/getRobotVersion`

Пример запроса: опущен

Пример ответа:

```json
{

"code":200, "data":"version 12313", "message":"success"

}
```

### Получите основную информацию об автомобиле

URL-адрес запроса

`http://<ip>:9998/tool/rob/getVehicle`

Пример запроса: опущен

Пример ответа:

```json
{

"code":200, "message":"success", "data":{

"vehicleId":"vehicle UUID",

"mapId":"current map", "vehicleType":"vehicle model"

}

}
```

### Получить пакет карты с хоста

URL-адрес запроса

`http://<ip>:9998/tool/rob/getRunMapZip`

Параметры запроса:

Тип контента: приложение/json

|  |  |  |
| --- | --- | --- |
| Требуется | Параметры | Описание |
| Да | Имя | Название карты |

Пример ответа:

Возвращает файл с типом содержимого: application/zip.

```json
        "y": -0.57843643426895142,

        "yaw": 1.6737515926361084

```

- 1\. **Переезд(Выйти в Интернет)**

URL-адрес запроса

`http://<ip>:9998/tool/rob/relocate`

Параметры запроса:

Тип контента: приложение/json

|  |  |  |
| --- | --- | --- |
| Требуется | Параметры | Описание |
| Да | х | Позиция X (метры) |
| Да | й | Позиция Y (метры) |
| Да | угол | – |
| Да | шумныйX | Шумность (метры) |
| Да | шумный | Шумность (метры) |
| Да | шумныйугол | Шумность (метры) |

Пример запроса: опущен

Пример ответа:

```json
{

"code":200, "message":"success",

}
```

## Описание интерфейса

Красная строка меню: нажмите, чтобы ввести сведения о работе соответствующей функции, например, начало картографии, перемещение, управление картой и т. д.

Синий прямоугольник: отображается с помощью загруженной 2D-карты контуров и значка отображения местоположения автомобиля.

Зеленое поле: отображает RGB-изображение камеры верхнего вида в реальном времени.

Желтое поле: отображение силы сигнала перемещения в реальном времени (зеленый: отличная сила, синий: хорошая сила, красный: плохая сила); отображение в реальном времени координат текущего положения автомобиля, текущей скорости движения автомобиля и координат положения мыши

## 1. Начните создавать карту

Нажмите кнопку **Начать создание карты** в веб-интерфейсе, как показано на рисунке ниже:

Введите название карты (имя карты состоит из букв, цифр и подчеркиваний и не может начинаться с цифр), как показано на рисунке ниже:

Это страница, с которой можно начать картографирование

Затем переместите AGV для сканирования карты в зоне действия. После завершения сканирования щелкните строку меню, чтобы завершить построение карты. Пожалуйста, подождите терпеливо (время ожидания зависит от размера области сканирования).

Если сопоставление выполнено успешно, данные сопоставления можно увидеть в элементе управления сопоставлением.

инкрементное картографирование

Функция: добавление новых областей карты на основе существующих карт, например расширение карты и дополнение слепых зон.

Предварительное условие: при создании инкрементных карт вам необходимо убедиться, что веб-страница завершила перемещение и что вы переключились на карту, которую необходимо добавить.

(На данный момент выдается ручной HTTP-запрос)

Адрес [http://192.168.100.201:9998/tool/rob/sendCMD](http://192.168.100.201:9998/tool/rob/sendCMD "http://192.168.100.201:9998/tool/rob/sendCMD")

Параметр Json

```json
{

    "cmd":905, // 904 starts incremental mapping; 905 stops incremental mapping

    "attach":{

        "sub_command":0,

        "param":""

}

}
```

## 2. Управление картой

![2. Иллюстрации, связанные с управлением картой.](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/9b855ff4-d2b8-4a0b-9031-3b1343a848df.avif) Рисунок 2 Схема управления картой

После завершения создания карты нажмите **Управление картой** в строке меню. Вы можете увидеть имя карты, которую вы только что создали, в списке карт хостов. Нажмите кнопку «Применить», чтобы переключиться на карту. Аналогично, если нужно переключить несколько региональных карт, вам также необходимо использовать функцию управления картами для переключения. Нажмите кнопку «Удалить», чтобы удалить соответствующую карту; **Применить карту.** После нажатия кнопки «Применить» обязательно перезапустите хост, прежде чем карта будет применена и вступит в силу, иначе перемещение будет невозможно.

Перемещение QR-кода

При построении карты вы можете вставить код Apritag в некоторые места для перемещения. Когда полный QR-код появится в перспективе устройства, оно автоматически определит, на какой карте и в каком месте находится код, автоматически переключит карту и переместится.

Сайт генерации QR-кода [https://chaitanyantr.github.io/apriltag](https://chaitanyantr.github.io/apriltag "https://chaitanyantr.github.io/apriltag")

## 7. Обновление прошивки

Просмотреть локальный пакет: импортируйте локальную прошивку, которую необходимо обновить (шасси, привод, прошивку камеры).

Пакет восстановления: Запишите версию прошивки до последнего обновления. Если вам нужно вернуться назад, просто нажмите на нее.

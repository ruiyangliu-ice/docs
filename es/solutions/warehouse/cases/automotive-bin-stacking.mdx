## Antecedentes del proyecto

En el centro logístico de una planta de fabricación de automóviles se requiere una gestión automatizada del apilamiento de jaulas que contienen piezas estampadas. Cada jaula tiene una altura de una sola capa de 1,6 metros y debe apilarse en cuatro capas, con una altura total de 6,4 metros. El peso de las piezas estampadas cargadas en cada jaula supera la tonelada, por lo que este proyecto impone requisitos extremadamente altos en cuanto a **acoplamiento de alta precisión** y seguridad durante el proceso de apilamiento.

## Solución

MRDVS proporcionó una **solución de apilamiento de contenedores con visión 3D** para este proyecto. Una cámara M4 Pro instalada en la carretilla elevadora inteligente detecta la pose relativa de cada jaula y guía la carretilla elevadora durante el apilado de alta precisión. El sistema monitorea y ajusta la posición y el ángulo en tiempo real para mantener el **espacio máximo entre jaulas por debajo de 5 mm** y garantizar un apilamiento preciso y seguro.

> **Tecnología clave**

**• Apilamiento multicapa**: El sistema admite el apilamiento de cuatro capas de jaulas de una sola capa de 1,6 metros de altura, con una altura total de apilamiento de 6,4 metros.

**• Atraque de alta precisión**: La postura relativa del montacargas y la jaula de material se calcula en tiempo real a través de la cámara 3D para garantizar que el error de identificación de las patas de la jaula de material durante el proceso de apilamiento sea inferior a 5 mm, lo que garantiza la estabilidad del apilamiento.

**• Carga de peso elevado**: la solución puede manejar de forma segura jaulas que pesen más de 1 tonelada y la precisión del acoplamiento se controla a nivel milimétrico para garantizar un apilamiento seguro bajo cargas elevadas.

**• Protección de seguridad y alarmas**: El sistema monitorea el proceso de apilamiento para detectar condiciones anormales. Si la desviación de postura detectada excede el rango seguro, el montacargas retrocede y repite la detección y el acoplamiento. El sistema también puede emitir una **alarma de condición anormal** para que el operador pueda intervenir antes de que se desarrolle una situación peligrosa.

## Visualización de resultados

A través de la solución de apilado de cestas con visión 3D de Meyer Microvision, el centro logístico del fabricante de automóviles ha realizado un proceso de apilado automatizado eficiente y seguro. Durante el proceso de apilado de cada capa de jaulas, el sistema garantiza una precisión milimétrica y completa de forma segura y estable el apilado de varias capas, lo que mejora la eficiencia del almacenamiento y reduce eficazmente la intervención manual.

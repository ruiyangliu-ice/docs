## 1. Introducción

### 1.1 Objeto y alcance

Este documento proporciona las especificaciones de la cámara H3310 y detalles de diseño para los desarrolladores que integran y utilizan el producto.

### 1.2 Terminología

| Línea de base | En una cámara de profundidad estéreo, la línea de base es la distancia horizontal entre las dos cámaras. Más precisamente, es la distancia entre sus ejes ópticos y normalmente se mide en milímetros (mm). |
| --- | --- |
| Profundidad | Al igual que una transmisión de video en color o monocromática, una transmisión de profundidad asigna un valor a cada píxel. Este valor representa la distancia del píxel a la cámara. |
| campo de visión | Campo de visión. La extensión angular de una escena que la cámara puede capturar. Incluye FOV horizontal (HFOV), FOV vertical (VFOV) y FOV diagonal (DFOV). |
| Cámara de infrarrojos | Una cámara de infrarrojos. En este producto, captura principalmente imágenes en el espectro del infrarrojo cercano. |
| Cámara RGB | Una cámara a color. |
| Cámara de profundidad | Una cámara con un módulo de imágenes de profundidad que normalmente consta de un proyector de puntos, una cámara de infrarrojos y un procesador de profundidad. Su módulo de imágenes en color suele ser una cámara RGB. |
| SoC | Un sistema en chip que procesa datos de imágenes, calcula la profundidad y ejecuta algoritmos de aplicación. |

### 1.3 Tecnología de luz estructurada

La luz estructurada binocular es una tecnología común de medición de visión 3D que combina visión estéreo con proyección de luz estructurada. Utiliza dos cámaras y un proyector de luz estructurada para capturar la forma y profundidad 3D de un objeto.

El proceso funciona de la siguiente manera:

1. Proyección de luz estructurada: un proyector proyecta luz estructurada sobre el objeto objetivo. Una fuente de luz infrarroja o visible normalmente produce la luz, que está especialmente codificada o modulada mediante un patrón de cuadrícula o moteado. La luz proyectada crea patrones brillantes, oscuros o distorsionados en la superficie del objeto.

2. Imagen estéreo: mientras se proyecta la luz estructurada, dos cámaras capturan el objeto objetivo simultáneamente. Las cámaras están separadas por una distancia fija, similar a la distancia entre los ojos humanos. Registran tanto el patrón proyectado como la apariencia del objeto.

3. Captura de imágenes: Las cámaras capturan imágenes de izquierda y derecha. La distorsión o desplazamiento del patrón proyectado en estas imágenes codifica la información de profundidad del objeto.

4. Procesamiento de imágenes: las diferencias entre las imágenes izquierda y derecha se utilizan para calcular la profundidad de la superficie. La combinación de estas diferencias con el patrón proyectado produce un modelo de nube de puntos 3D del objeto.

5. Medición y reconstrucción: la nube de puntos 3D resultante se puede utilizar para medir, analizar o reconstruir la forma y las dimensiones del objeto.

## 2. Descripción general del producto

La H3310 es una cámara industrial de luz estructurada RGB-D que pesa 400 g. Puede montarlo en un robot compuesto o en un brazo robótico pequeño. A una distancia de reconocimiento de 30-60 cm, proporciona datos 3D de alta precisión y datos RGB de alta resolución. Su plataforma informática integrada ejecuta algoritmos de aplicación de IA de forma independiente para tareas de reconocimiento, inspección y posicionamiento de robots industriales.

### 2.2 Características

- Plataforma informática integrada: proporciona 6.0 TOPS de potencia informática y ejecuta algoritmos de aplicaciones sin una CPU externa.

- Salida de datos 3D: combina imágenes de luz estructurada con algoritmos de IA para proporcionar datos 3D de alta precisión.

- Diseño compacto: pesa 400 gy proporciona interfaces abiertas para su implementación.

- Producción en masa: admite una producción estable y de gran volumen.

## 7. Preguntas frecuentes

| Número | Problema | Resolución |
| --- | --- | --- |
| 1 | La cámara no responde cuando intentas abrirla | Compruebe si el firewall está deshabilitado. Debes desactivar el firewall. |
| 2 | El host detecta múltiples direcciones IP | La cámara adquirió una dirección IP que ya está en uso en la LAN. Seleccione la dirección IP requerida para abrir la cámara. |
| 3 | Los datos de la cámara son inestables después de abrir la cámara | Utilice un cable Gigabit Ethernet. Los datos pueden ser inicialmente inestables con un cable Ethernet de 100 Mbps. |
| 4 | Ubicación de instalación del software | La instalación del software en la unidad C predeterminada puede requerir configuración de permisos. |

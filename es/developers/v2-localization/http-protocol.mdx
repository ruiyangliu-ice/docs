## Recuperar información global

### Actualizar programa principal

Solicitar URL

`http://<ip>:9998/tool/rob/quiet_update`

Ejemplo de solicitud:

```json
{

"url":"http://192.168.2.179:8080/hontai_2.1.20211114.zip"

}
```

### Establecer altura de la cámara

Solicitar URL `http://<ip>:9998/tool/rob/setHeightRequest` Ejemplo:

```json
{

"height": 1.3

}
```

Ejemplo de respuesta:

```json
{

"code":200, "message":"success",

}
```

- 1\. **Reubicación (Conectarse en línea)**

`http://<ip>:9998/tool/rob/relocate`

Parámetros de solicitud:

Tipo de contenido: aplicación/json

|  |  |  |
| --- | --- | --- |
| Requerido | Parámetros | Descripción |
| Sí | x | Posición X (metros) |
| Sí | y | Posición Y (metros) |
| Sí | ángulo | – |
| Sí | ruidosoX | Ruidoso (metros) |
| Sí | ruidosoY | Ruidoso (metros) |
| Sí | ángulo ruidoso | Ruidoso (metros) |

Ejemplo de solicitud: omitido

Ejemplo de respuesta:

```json
{

"code":200, "message":"success",

}
```

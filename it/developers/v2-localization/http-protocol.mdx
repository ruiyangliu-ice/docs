## Recupera informazioni globali

### Aggiorna il programma principale

Richiedi URL

`http://<ip>:9998/tool/rob/quiet_update`

Esempio di richiesta:

```json
{

"url":"http://192.168.2.179:8080/hontai_2.1.20211114.zip"

}
```

### Imposta l'altezza della telecamera

Richiedi URL `http://<ip>:9998/tool/rob/setHeightRequest` Esempio:

```json
{

"height": 1.3

}
```

Esempio di risposta:

```json
{

"code":200, "message":"success",

}
```

`http://<ip>:9998/tool/rob/getVehicle`

Esempio di risposta:

```json

### Ottieni il pacchetto mappa dall'host

Richiedi URL

`http://<ip>:9998/tool/rob/getRunMapZip`

Richiedi parametri:

Tipo di contenuto: application/json

|  |  |  |
| --- | --- | --- |
| Obbligatorio | Parametri | Descrizione |
| Sì | Nome | Nome della mappa |

Esempio di risposta:

Restituisce un file con tipo di contenuto: application/zip

- 1\. **Trasloco (vai online)**

Tipo di contenuto: application/json

|  |  |  |

| Obbligatorio | Parametri | Descrizione |

| Sì | sì | Posizione Y (metri) |

| Sì | rumorosoX | Rumoroso (metri) |
| Sì | rumorosoY | Rumoroso (metri) |
| Sì | angolo rumoroso | Rumoroso (metri) |

Esempio di richiesta: omesso

"code":200, "message":"success",

}
```

|  |  |  |
| --- | --- | --- |
| Obbligatorio | Parametri | Descrizione |
| Sì | id | Nome della mappa |

## 1. はじめに

### 1.1 目的と範囲

このドキュメントでは、S10 カメラの仕様と、製品を統合して使用する開発者向けの設計の詳細について説明します。

### 1.2 用語

| RGBDカメラ |同期され、空間的に位置合わせされた RGB および深度画像を出力するカメラ |
| --- | --- |
|深さ |カラーまたはモノクロ ストリームに似たビデオ ストリーム。各ピクセルはカメラからの距離を表します。
|視野 |視野: 水平 (HFOV)、垂直 (VFOV)、対角 (DFOV) の視野を含む、カメラによってキャプチャされたシーンの角度範囲。
|赤外線カメラ |赤外線カメラ。本製品では主に近赤外線帯域を捉えます。
| RGBカメラ |カラーカメラ |
|深度カメラ |通常、深度モジュールにドット プロジェクター、IR カメラ、および深度プロセッサが含まれるカメラ。そのカラー モジュールは通常、RGB カメラです。
| SoC |画像処理、深度計算、アプリケーション アルゴリズムに使用されるシステム オン チップ |
| DTOF |飛行時間測定に基づく距離センシング技術 |

### 1.3 動作原理

この製品は、長距離、良好な精度、低コスト、強力な耐干渉性、安定したデータを備えた DTOF 3D 深度カメラです。統合された RGB モジュールがカラー画像をキャプチャします。カメラは内部で RGB イメージと深度イメージを調整します。

## 2. 製品概要

このコンパクトな産業用 RGBD カメラは、0.3 m ～ 8 m の深度データを提供します。 RGB イメージと深度イメージを空間的および時間的に位置合わせします。統合された SoC は、深度、強度、RGB、点群、障害物検出の結果を出力します。イーサネットに加えて、2 つの IO 入力と 2 つの IO 出力が 4 つの障害物回避ゾーンを切り替え、緊急停止、減速、安全状態信号を出力します。

### 2.2 機能

- 0.3～8mの広い測定範囲

## 6. クイックスタート

### 6.1 接続テスト

<Frame>
  <img src="/images/legacy/cb764c04-73c2-4cdd-9381-98fe1c201737/24cb2351-298f-4381-8170-c34cf1444269.avif" alt="接続テスト" />
</Frame>

1. カメラを 24V/2A DC 電源に接続します。青色の電源インジケーターがゆっくり点滅することで、電源が正常にオンになっていることを確認できます。

2. カテゴリ 6 イーサネット ケーブルを使用してカメラをコンピュータに接続します。

3. 工場出荷時のデフォルトのカメラ IP アドレスは 192.168.100.82 です。

4. コンピュータの IP をカメラのサブネットに設定し、コンピュータのファイアウォールを無効にします。

5. ホスト ソフトウェアを実行してカメラ画像を表示します。

6. MRDVS セールスまたはテクニカル サポートに連絡して、ホスト ソフトウェアを入手します。

### 6.2 IO 配線

|インターフェース |タイプ |メモ |
| --- | --- | --- |
| IN1 |入力 |アクティブハイのディスクリート入力。 24V/オープン入力対応 |
| IN2 |入力 |アクティブハイのディスクリート入力。 24V/オープン入力対応 |
| OUT1 |出力 |グランド/オープンディスクリート出力。 24Vプルアップをサポート |
|アウト2 |出力 |グランド/オープンディスクリート出力。 24Vプルアップをサポート |
|コム |グランド |ディスクリート出力の共通グランド。 IN1 を IN_COM1 とともに使用する |

## 8. トラブルシューティング

|番号 |問題 |解像度 |
| --- | --- | --- |
| 1 |カメラを開いたときに反応しません |ファイアウォールが無効になっていることを確認します。
| 2 |ホスト ソフトウェアが複数の IP アドレスを検出します。 LAN 上の別のデバイスが同じ IP を使用します。必要な IP アドレスを選択します |
| 3 |開いた後のカメラデータが不安定です |ギガビット イーサネット ケーブルを使用します。データは、ファースト イーサネット経由では最初は不安定になる可能性があります。
| 4 |ソフトウェアのインストール場所 |ドライブ C へのデフォルトのインストールは、権限設定の影響を受ける可能性があります。

<Step title="パッケージを検査する">

カメラを受け取ったら、付属品や付属品に欠品や破損がないか確認してください。不足しているものや破損しているものがある場合は、MRDVS テクニカル サポートにお問い合わせください。

</Step>

<Step title="M4 Pro カメラを接続する">

以下のように電源とホストコンピュータを接続します。

1. 取り付けまたは使用中にレンズ表面を遮らないでください。左の画像は正しい取り付けを示し、右の画像は間違った取り付けを示しています。

<Frame>
  <img src="/images/external/a2d017b8fff069857a6df0a3.avif" alt="M4 Pro カメラの正しい取り付けと間違った取り付け" />
</Frame>

<Frame>
  <img src="/images/external/84c19b4c11995706c8cddd81.avif" alt="M4 Pro カメラ接続図" />
</Frame>

2. 付属の5ピンケーブルをカメラに接続し、24V / 3Aを供給します。電力が正常な場合、青色の電源インジケーターが 2 秒ごとにゆっくり点滅します。

3. 以下に示すように、カテゴリ 6 以上のイーサネット ケーブルを使用してカメラをコンピュータに接続します。

<Frame>
  <img src="/images/legacy/2e3dae80-ce2f-4248-bcb2-5d07b6ab805a/c4c0a700-2858-4222-92c5-9ee61197ae57.avif" alt="M4 Pro カメラのイーサネット接続" />
</Frame>

</Step>

<Step title="電源">

カメラはDC電源を使用します。信号線の定義については、[M4 Pro 製品ユーザーガイド](/ja/products/m-series/m4-pro-user-guide "M4 Pro 製品ユーザーガイド") のセクション 3.7「製品インターフェイス」を参照してください。

</Step>

<Step title="IPアドレスを設定する">

1. カメラの工場出荷時のデフォルト IP アドレスは 192.168.100.82 です。コンピューターを 192.168.100.XXX サブネットに設定します。

2. LxCameraViewer がカメラを開くには、カメラとコンピュータが同じサブネット上にある必要があります。

3. カメラを接続するときにネットワーク ファイアウォールを無効にします。

</Step>

<Step title="コンピュータネットワークを設定する">

**コントロール パネル**を開き、**ネットワークとインターネット** → **ネットワークと共有センター** → **アダプター設定の変更** → **イーサネット** → **インターネット プロトコル バージョン 4 (TCP/IPv4)** を選択します。 [**インターネット プロトコル バージョン 4 (TCP/IPv4) のプロパティ**] ダイアログで、**次の IP アドレスを使用する** を選択します。カメラのサブネット 192.168.100.XXX のアドレスを入力し、サブネット マスクを 255.255.0.0 に設定します。

<Frame>
  <img src="/images/external/276992291dbad945ddfdb9f7.avif" alt="WindowsのIPv4ネットワーク設定" />
</Frame>

</Step>

<Step title="LxCameraViewer をダウンロードして画像を表示する">

1. LxCameraViewer を使用して画像を表示し、カメラ画像パラメータを設定します。

2. LxCameraViewer を使用して、カメラの内蔵障害物回避、パレット、ナビゲーション、および把握アルゴリズムを設定します。

3. お使いのシステム用の LxCameraViewer バージョンをダウンロードします。ホスト アプリケーションは Windows では利用できますが、Linux では利用できません。 ROS を使用してサンプルを実行し、rviz でイメージを表示します。

- GitHub: [Lanxin-MRDVS/カメラSDK](https://github.com/Lanxin-MRDVS/CameraSDK)

- ソフトウェア: [ソフトウェアのダウンロード](https://zcnt5g3o2989.feishu.cn/wiki/Q8IYwGiGaiIvIyktiavczymanOe?from=from_copylink "ソフトウェアのダウンロード")

</Step>

<Step title="LxCameraViewerを起動する">

1. インストーラーをダブルクリックしてLxCameraViewerをインストールします。次に、コンピュータの IP アドレスを構成し、ネットワーク ファイアウォールを無効にします。

<Frame>
  <img src="/images/legacy/e00efd7e-06aa-4de8-9178-7e7e8e92c011/0629f2ec-63dc-4388-ac39-b78e306030df.avif" alt="LxCameraViewer インターフェース" />
</Frame>

2. LxCameraViewer をダブルクリックし、左側のデバイスリストにカメラ ID が表示されるかどうかを確認します。

- デバイスが表示されない場合は、カメラのインジケーターが 2 秒ごとに点滅するかどうかを確認します。

- 正常に点滅する場合は、ネットワーク接続のトラブルシューティングを行ってください。

- 点滅せずに点灯したままの場合は、MRDVS テクニカル サポートにお問い合わせください。

- オフになっている場合は、電源が 24V / 3A の要件を満たしていることを確認してください。

<Frame>
  <img src="/images/legacy/fcd6242d-8a7a-41e7-8cc0-bef2f10bec65/a0cf4ddc-5e81-4e85-85ee-3fef57c20c2e.avif" alt="LxCameraViewer デバイスリスト" />
</Frame>

3. [**開く**] を選択してストリームを開始します。ストリーミングが開始されると、ボタンは **閉じる** に変わります。深度、振幅、点群ビューがデフォルトで開きます。 RGB ビューを開くには、そのチェックボックスを選択します。

<Frame>
  <img src="/images/legacy/e5741b74-091f-4bbe-81b8-374b66f6fd68/7fe62674-9ea9-4bd6-bb35-7db6bc2c7bdd.avif" alt="LxCameraViewer ストリーム コントロール" />
</Frame>

4. 右側のカメラパラメータ設定を選択して、使用可能なパラメータを展開します。

<Frame>
  <img src="/images/legacy/726a4493-2e26-4dfa-8458-c95c4ccac5d9/7c7194ea-18ac-434e-9ddb-a67d02000d23.avif" alt="LxCameraViewerのカメラ設定" />
</Frame>

5. [**アルゴリズムを適用**] を選択して、アプリケーション アルゴリズムを選択または切り替え、動作モードを選択します。

<Frame>
  <img src="/images/legacy/ed478a24-b097-4a66-98ca-ff842bca6042/c7d959c9-e656-4ba9-b0b8-ab74487456c6.avif" alt="LxCameraViewerのアルゴリズム設定" />
</Frame>

6. これらの手順では、LxCameraViewer の基本的な操作について説明します。パラメータはバージョン間で若干異なる場合があります。最新の手順については、[LxCameraViewer ホスト アプリケーション ユーザー ガイド](https://zcnt5g3o2989.feishu.cn/wiki/IHK1wiLy2iIccEkgZ3pc74jhnAf?from=from_copylink "LxCameraViewer ホスト アプリケーション ユーザー ガイド") を参照してください。

## カメラに関する注意事項

**これらの指示に従ってください。不適切な操作を行うと、内部コンポーネントが損傷する可能性があります。**

1. カメラを落としたり、機械的ストレスを与えたりしないでください。

2. 内部カメラファイルを変更しないでください。永久的な損傷を引き起こす可能性があります。

3. 長時間使用するとカメラが温かくなることがありますが、画像や距離の精度には影響しません。パッシブ冷却を使用します。カメラと金属製取り付けプレートの間の接触面積を増やして、熱放散を改善します。

4. レンズには触れないでください。指紋が付着すると画質が低下する場合があります。

5. カメラが検出されない場合は、電源が 24V / 3A を供給していること、およびイーサネット ケーブルが損傷していないことを確認します。

**カメラの安全性と取り扱い:**

1. 外部に損傷がある場合は、製品の電源を入れないでください。

2. 製品のいかなる部分も開かないでください。許可なく分解すると、アフター修理サービスが無効になります。

3. 目に見えないレーザー放射に注意してください。カメラを直射日光に向けないでください。

4. 製品の改造や修理はしないでください。コンプライアンスや安全基準に違反する可能性があります。

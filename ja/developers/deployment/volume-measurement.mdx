## 1. はじめに

体積検出アルゴリズムは Meyer Micro Vision (MRDVS) によって開発され、特に体積測定シナリオで使用されます。これは、物流、速達、石炭ヤード、倉庫、その他のシナリオで自社開発の M シリーズ カメラで使用されています。体積検出アルゴリズムはカメラ内で実行されるため、外部の産業用コンピューターは必要ありません。このアルゴリズムは静的検出と動的検出をサポートしています。

このソフトウェアは、ユーザーがカメラの外部パラメータを調整し、検出範囲を描画するだけで、単一または複数の物体を検出および測定できます。外部パラメータのキャリブレーションは、キャリブレーション結果を自動的に取得し、マルチカメラのスプライシング検出をサポートします。

この記事では、Meyer マイクロビジョンのボリューム検出機能の導入と使用方法を主に紹介します。

## 2. ボリューム検出アプリケーションのシナリオ

### 2.1 物流業界

1.物品の体積や重量などの情報を取得することで、きめ細かな業務を実現することができ、車両積載率の向上、車両コスト、輸送費、人件費の削減など、全工程にわたるコスト削減と効率化を実現します。

2. ワンクリックで商品/箱の体積と重量を素早く検出し、運賃を計算できます。フロアスケールと併用すると、トラックやコンテナトラックを効果的に測定して、車両全体が重量超過になっていないことを確認できます。

### 2.2 エクスプレス業界

速達便の発送および受け取りの際、インテリジェントな体積測定装置を利用すると、商品の体積や重量などの情報を迅速に取得できます。データは社内システムに接続されているため、検索や確認が容易です。これにより、データ損失の問題が回避されるだけでなく、手作業による投資が効果的に削減され、作業効率が向上します。

### 2.3 石炭鉱業

大きな石炭鉱石の山の体積を測定する従来の方法では、通常、トータル ステーションと RTK 単一点測定法が使用され、特徴点を手動で正確に抽出する必要があります。しかし、大きな石炭鉱石の山は登るのが不便であるため、正確な特徴点を抽出することは非常に困難です。一方、一点計測法ではデータ密度が低く、石炭鉱石の山の体積を正確に求めることができないため、作業効率が非常に低くなります。

### 2.4 倉庫業

物流や倉庫、さらには一般産業においてもAGVが物品を運びますが、入庫時や入庫時などに物品の体積や重量を確認する必要があります。

### 3. 問題を解決する

物流や倉庫などの産業では、品物の量と数量を把握し、品目の入退場を把握することが生産在庫管理にとって非常に重要です。しかし、従来のソリューションは、技術的手段の欠如、大きな測定誤差、手動管理の効率の低さ、監視の難しさなどの問題に常に直面しています。

Meyer Micro Vision ToF カメラのボリューム検出には、レーダー、構造化光、2D カメラよりも多くの利点があります。 Meyer Micro Vision カメラは導入が簡単で、必要なのはカメラのファームウェア ブラケット、電源コード、ネットワーク ケーブルのみです。このアルゴリズムは、産業用コンピューターの冗長構成を必要とせずにカメラ内で実行されます。レーダーよりも使いやすく、コストも低くなります。

### 7.3 体積検出外部パラメータの校正

1. カメラの IP 構成が完了し、ファイアウォールが閉じられたら、展開の 2 番目のステップに進みます。ソフトウェアを開いて、アルゴリズムの外部パラメーターを調整します。特定の基準地面とカメラの設置角度の偏差を校正する必要があります。

2. カメラ設定をクリックし、パラメータ調整インターフェイスに切り替えます。外部パラメータキャリブレーションを実行する前に、x、y、z、roll、pitch、yaw が 0 であることを確認してください。0 でない場合は、すべてのパラメータを 0 に戻し、[**保存して送信**] ボタンをクリックする必要があります。パラメータが正常に保存されたら、外部パラメータを再調整します。

</Frame>

3. パラメータがすべて 0 であることを確認した後、[**外部パラメータ校正**] ボタンをクリックします。キャリブレーションが成功すると、ソフトウェアによりプロンプト ボックスがポップアップ表示されます。右側のパラメータ列に、調整されたカメラパラメータが表示されます。

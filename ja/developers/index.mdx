この領域では、製品カタログ内でソフトウェア操作、SDK 開発、ROS インターフェイス、およびアルゴリズム展開が混在することを避けるために、エンジニアリング タスクごとにドキュメントを整理します。

## 注目のガイド

<Card title="V2 で外部 LiDAR を使用する" icon="radio-tower" href="/ja/developers/v2-localization/external-lidar">
外部 LiDAR データを受信し、ROS 1 または ROS 2 レーザー トピックを確認するように V2 ローカリゼーション モジュールを構成します。
</Card>

## 移行範囲

- ソフトウェアツール: LxCameraViewer および一般的なソフトウェア操作。
- カメラ SDK: SDK マニュアル、Linux サンプル、および JSON 通信プロトコル。
- ROS 統合: クイックスタートと ROS 1 および ROS 2 インターフェイス。
- V2 測位モジュール: ユーザーマニュアル、UDP、HTTP、および外部レーザープロトコル。
- アルゴリズムとソリューションの展開: 障害物回避、パレット認識、検出機能、体積測定。

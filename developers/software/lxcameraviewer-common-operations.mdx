本页汇总 LxCameraViewer 中常用的设备配置与维护操作。有关软件安装、设备连接和完整界面说明，请参阅 [LxCameraViewer 上位机软件使用手册](/developers/software/lxcameraviewer)。

<Warning>
  升级相机固件或应用算法前，确认升级包与相机型号匹配。升级过程中不要断开电源或网络连接。
</Warning>

## 网络与接口

### 修改相机 IP 地址

在 **其他工具** 中打开 **IP 配置工具**，选择目标相机后修改 IP 地址、子网掩码或网关，并确认主机与相机处于同一网段。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-camera-ip-modification-method.avif" alt="IP 配置工具中的相机网络参数设置" />
</Frame>

### 配置 CAN 波特率和 CAN ID

在 **基础工具** → **硬件接口** 中修改 CAN 波特率和 CAN ID。接入同一 CAN 总线的设备必须使用一致的波特率。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-modifying-can-baud-rate-and-can-id-methods.avif" alt="LxCameraViewer 中的 CAN 波特率和 CAN ID 设置" />
</Frame>

## 版本与升级

### 查看软件、固件和算法版本

打开相机后，在设备信息或软件的版本信息区域查看相机固件、LxCameraViewer 和应用算法版本。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-viewing-methods-of-firmware-version-pc-software-version-and-algorithm-versio.avif" alt="LxCameraViewer 中查看固件、软件和算法版本" />
</Frame>

### 升级相机固件或应用算法

在 **基础工具** → **功能设置** 中选择固件升级或应用算法升级，并按界面提示选择交付的升级文件。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-camera-firmware-upgrade-and-application-algorithm-upgrade-methods.avif" alt="LxCameraViewer 中升级相机固件或应用算法" />
</Frame>

## 参数与成像

### 导入或导出成像参数

在 **基础工具** → **功能设置** 中导出当前相机配置，或从文件加载已保存的配置。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-import-and-export-methods-of-camera-imaging-parameters.avif" alt="LxCameraViewer 中导入和导出相机成像参数" />
</Frame>

### 使应用算法常开

在 **应用算法** 中配置工作模式，使相机在与上位机断开连接后继续输出已启用的算法结果。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-applying-algorithm-to-set-normally-open-method.avif" alt="LxCameraViewer 中配置应用算法常开" />
</Frame>

### 获取相机内参

在 **基础工具** → **功能设置** 中读取相机内参。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-the-method-of-obtaining-internal-parameters-of-the-camera.avif" alt="LxCameraViewer 中获取相机内参" />
</Frame>

### 启用 RGB-D 对齐

在 **基础工具** → **3D 设置** 中启用 RGB-D 对齐。

<Frame>
  <img src="/images/developers/software-lxcameraviewer-common-operations-rgbd-alignment-opening-method.avif" alt="LxCameraViewer 中启用 RGB-D 对齐" />
</Frame>

本页汇总 LxCameraViewer 中常用的设备配置与维护操作。有关软件安装、设备连接和完整界面说明，请参阅 [LxCameraViewer 上位机软件使用手册](/developers/software/lxcameraviewer)。

<Warning>
  升级相机固件或应用算法前，确认升级包与相机型号匹配。升级过程中不要断开电源或网络连接。
</Warning>

## 网络与接口

### 修改相机 IP 地址

在 **其他工具** 中打开 **IP 配置工具**，选择目标相机后修改 IP 地址、子网掩码或网关，并确认主机与相机处于同一网段。

<Frame>
  <img src="/images/legacy/ff10582b-cc86-4cb1-9819-131423deb0c6/880b3a93-7d1b-49b2-b866-881e3c5e7aa5.avif" alt="IP 配置工具中的相机网络参数设置" />
</Frame>

### 配置 CAN 波特率和 CAN ID

在 **基础工具** → **硬件接口** 中修改 CAN 波特率和 CAN ID。接入同一 CAN 总线的设备必须使用一致的波特率。

<Frame>
  <img src="/images/legacy/4301be14-b4fd-42af-a9b3-3a4afe54a511/b25a4b59-ff5a-4e54-b034-6289d256719c.avif" alt="LxCameraViewer 中的 CAN 波特率和 CAN ID 设置" />
</Frame>

## 版本与升级

### 查看软件、固件和算法版本

打开相机后，在设备信息或软件的版本信息区域查看相机固件、LxCameraViewer 和应用算法版本。

<Frame>
  <img src="/images/legacy/d70a69e1-9c3a-455f-bc49-8045c46e38e8/d19875bb-718d-4a78-ae85-6cd53097b8f4.avif" alt="LxCameraViewer 中查看固件、软件和算法版本" />
</Frame>

### 升级相机固件或应用算法

在 **基础工具** → **功能设置** 中选择固件升级或应用算法升级，并按界面提示选择交付的升级文件。

<Frame>
  <img src="/images/legacy/70e59d2f-b849-4022-b900-ba8b2c588bb3/22ed393c-f0ca-47ca-940e-7e17c5ce5d46.avif" alt="LxCameraViewer 中升级相机固件或应用算法" />
</Frame>

## 参数与成像

### 导入或导出成像参数

在 **基础工具** → **功能设置** 中导出当前相机配置，或从文件加载已保存的配置。

<Frame>
  <img src="/images/legacy/19002bea-67ab-4f9f-88f4-19e7ffed8a16/ead9ca29-0e1c-40a3-a334-ea9fad945ce1.avif" alt="LxCameraViewer 中导入和导出相机成像参数" />
</Frame>

### 使应用算法常开

在 **应用算法** 中配置工作模式，使相机在与上位机断开连接后继续输出已启用的算法结果。

<Frame>
  <img src="/images/legacy/caa7d264-7b6d-4f8a-bff9-91066d31b901/df5c1c6a-0fbf-4040-9747-9ebe33415871.avif" alt="LxCameraViewer 中配置应用算法常开" />
</Frame>

### 获取相机内参

在 **基础工具** → **功能设置** 中读取相机内参。

<Frame>
  <img src="/images/legacy/c220aab5-de16-46ad-b1ce-99eca73c5440/16e1f91f-fe41-465d-88d1-a96defd4e2cb.avif" alt="LxCameraViewer 中获取相机内参" />
</Frame>

### 启用 RGB-D 对齐

在 **基础工具** → **3D 设置** 中启用 RGB-D 对齐。

<Frame>
  <img src="/images/legacy/3fc563ac-2c69-414c-8a42-a6e4e7582bd3/9fcdf65b-7dcb-4b97-8f97-9e4471917546.avif" alt="LxCameraViewer 中启用 RGB-D 对齐" />
</Frame>

本页说明如何让 V2 定位模块订阅外部 LiDAR 的 ROS 激光话题。

<Info>
  外部 LiDAR 必须持续发布与配置一致的 `sensor_msgs/LaserScan` 消息；本页仅说明话题配置和订阅检查，不包含 LiDAR 驱动的安装与标定。
</Info>

## 配置定位激光话题

在本体配置中将定位激光设为 UDP 输入，并指定 LaserScan 话题：

```yaml
LxCamera_UploadScan: "/sim/scan"
```

将外部 LiDAR 的发布话题配置为同一名称，或按实际发布话题修改上述值。

## 验证 ROS 话题

确认 V2 定位节点订阅了该话题，并检查消息类型为 `LaserScan`。

<Tabs>
  <Tab title="ROS1">
    ```bash
    rostopic info /sim/scan
    ```

    预期可见消息类型 `sensor_msgs/LaserScan`，并且订阅者列表包含 `/lx_localization_node`。
  </Tab>
  <Tab title="ROS2">
    ```bash
    ros2 topic info /sim/scan
    ```

    预期可见消息类型 `sensor_msgs/msg/LaserScan`，并且订阅数量至少为 `1`。
  </Tab>
</Tabs>

<Tip>
  若订阅数量为 `0`，先检查话题名称、ROS 域配置和定位节点是否已启动；若发布数量为 `0`，检查外部 LiDAR 驱动。
</Tip>

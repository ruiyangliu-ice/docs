本页说明 V2 纯定位版本的 Web 界面操作。进行建图、切换地图或重定位前，确认车辆、相机和激光传感器已正常运行。

<Warning>
  地图切换、固件更新和本体程序重启会影响定位服务。请在车辆停止、现场安全且具备恢复条件时执行。
</Warning>

## 界面概览

| 区域 | 内容 |
| --- | --- |
| 菜单栏 | 建图、重定位、地图管理和维护功能。 |
| 地图视图 | 已加载的 2D 轮廓图和车辆定位图标。 |
| RGB 视图 | 顶视相机的实时 RGB 图像。 |
| 状态区 | 重定位信号强度、车辆坐标、速度和鼠标位置坐标。信号颜色从优到差依次为绿色、蓝色、红色。 |

## 建图

### 创建地图

<Steps>
  <Step title="开始建图">
    在 Web 界面点击 **开始建图**。

    <Frame>
      <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/4bb3eee5-75cd-4d29-b989-3c7c6d3fa88f.avif" alt="V2 纯定位界面的开始建图按钮" />
    </Frame>
  </Step>
  <Step title="输入地图名称">
    地图名只能包含英文字母、数字和下划线，且必须以英文字母开头。

    <Columns cols={2}>
      <Frame>
        <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/e400ede7-1ffa-49d1-b62a-e94692079e92.avif" alt="输入地图名称" />
      </Frame>
      <Frame>
        <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/248b2f19-f53b-4ab5-ad99-b826ab1b139d.avif" alt="开始建图页面" />
      </Frame>
    </Columns>
  </Step>
  <Step title="采集并结束建图">
    驾驶 AGV 覆盖目标区域。采集完成后点击 **结束建图** 并等待处理完成；处理时间与采集区域大小相关。
  </Step>
  <Step title="确认地图">
    建图成功后，在 **地图管理** 中确认新地图已出现。
  </Step>
</Steps>

### 增量建图

增量建图用于在已有地图上补充新区域。开始前，完成重定位并切换到需要补录的地图。

<Info>
  增量建图通过手动 HTTP 请求下发。接口地址为 [http://192.168.100.201:9998/tool/rob/sendCMD](http://192.168.100.201:9998/tool/rob/sendCMD)。
</Info>

<Tabs>
  <Tab title="开始增量建图">
    ```json
    {
      "cmd": 904,
      "attach": {
        "sub_command": 0,
        "param": ""
      }
    }
    ```
  </Tab>
  <Tab title="结束增量建图">
    ```json
    {
      "cmd": 905,
      "attach": {
        "sub_command": 0,
        "param": ""
      }
    }
    ```
  </Tab>
</Tabs>

结束后，新采集区域会写入当前地图；重启定位服务后生效。

## 重定位

### 手动重定位

点击 **重定位** 后，在地图视图中拖拽并提供车辆的大致位姿，使激光点云与轮廓图匹配。重定位成功后，**定位超时** 异常消失。

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/2e74ee5f-e192-4fce-8e8e-8f402988dcad.avif" alt="V2 纯定位的手动重定位界面" />
</Frame>

### AprilTag 重定位

建图时可在指定位置布置 AprilTag。设备视野内出现完整标签时，系统会识别其地图和位姿，并自动切换地图后重定位。

- 标签生成工具：[https://chaitanyantr.github.io/apriltag](https://chaitanyantr.github.io/apriltag)
- 同一设备的可见区域内，Tag ID 不得重复。
- Tag family 使用界面示例中的固定配置。

## 维护操作

### 更新固件

将本地固件包导入更新页面，可更新底盘、执行机构或相机固件。还原包记录上一次升级前的版本，可用于回退。

<Warning>
  使用普通上位机升级相机固件的方式不可回退。确认升级包和当前版本后再执行。
</Warning>

## 界面说明

红色菜单栏：点击可进入相应的功能操作详情，如开始建图、重定位、地图管理等

蓝色框：可见使用加载的2D轮廓图以及小车定位显示图标

绿色框:显示顶视相机实时RGB图

黄色框:实时显示重定位信号强度（绿色：强度优，蓝色：强度良好，红色：强度差）；实时显示小车当前的位置坐标，小车当前运动速度，以及鼠标位置坐标

此为开始建图页面

然后移动AGV在运行区域内扫图，扫图完成后点击菜单栏结束建图后，请耐心等待（等待时间与扫图区域大小相关）

建图成功时，在地图管理项中能看到本次建图数据

增量建图

功能 ： 在已有地图的基础上增加新的地图区域 如地图扩建 ，补充死角

前提 ： 增量建图的时候 需要确保网页端已经完成重定位 ，且确保已经切换到需要补录的地图

（暂时为手动Http请求下发）

地址 [http://192.168.100.201:9998/tool/rob/sendCMD](http://192.168.100.201:9998/tool/rob/sendCMD "http://192.168.100.201:9998/tool/rob/sendCMD")

参数 Json

```json
{

    "cmd":905, // 904为开始增量建图 905为结束增量建图

    "attach":{

        "sub_command":0,

        "param":""

}

}
```

## 二、地图管理

![二、地图管理相关图示](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/9b855ff4-d2b8-4a0b-9031-3b1343a848df.avif)图 2地图管理示意图

建图完成后，点击菜单栏**地图管理**，可以在主机地图列表中看到刚才建的地图名，点击应用按扭，则将切换使用该地图，同理，如果存在多个区域地图的切换，也需要使用地图管理功能进行切换，点击删除按扭可以删除对应的地图； **应用地图：**点击应用后且记一定要重启本体后，地图才会应用生效，否之会重定位不上等情况

## 三、重定位

点击重定位后可通过鼠标拖拽以激光点云和激光轮廓图的匹配提示给定大致重定位位姿，进行重定位；

<Frame>
  <img src="/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/2e74ee5f-e192-4fce-8e8e-8f402988dcad.avif" alt="三、重定位相关图示" />
</Frame>

图 4重定位功能示意图

重定位成功后**定位超时**异常消失。

二维码重定位

建图的时候可以选择在一些位置贴上Apritag码来进行重定位，当设备视角内出现完整的二维码后，会自动识别该码处于哪张地图以及位置，自动切换地图并重定位。

二维码生成网站[https://chaitanyantr.github.io/apriltag](https://chaitanyantr.github.io/apriltag "https://chaitanyantr.github.io/apriltag")

确保一个设备所处区域内的Tag ID不重复即可，TagFamily 为上图固定

## 七、固件更新

查看包本地：导入本地需要更新的固件（底盘、执行机构、相机固件）。

还原包:记录上次更新前固件版本，如需回退点击即可。

![七、固件更新相关图示](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/db55e5fb-c925-4105-bef1-84ac12f05997.avif)图 7固件更新

相机上位机固件升级

可使用普通上位机进行固件升级，此升级方式不可回退 需谨慎升级。![七、固件更新相关图示 2](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/4ec2ec9c-5285-4a4d-b91e-0fea462d9bed.avif)

点击执行相关操作，如需用上位机连接相机需关闭本体程序，不然相机被小车占用，无法用上位机与相机通讯 ![七、固件更新相关图示 3](/images/legacy/c450a63a-7b1c-4291-9e40-4cb352361b6e/6a00c674-61ac-4cde-8af7-c7e9389ccc6d.avif)

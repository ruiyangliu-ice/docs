## Retrieve Global Information

### Update Main Program

Request URL

`http://<ip>:9998/tool/rob/quiet_update`

Request Example:

```json
{

"url":"http://192.168.2.179:8080/hontai_2.1.20211114.zip"

}
```

### Set Camera height

Request URL `http://<ip>:9998/tool/rob/setHeightRequest` Example：

```json
{

"height": 1.3

}
```

Response Example：

```json
{

"code":200, "message":"success",

}
```

|  |  |  |

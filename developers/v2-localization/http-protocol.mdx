本页列出 V2 视觉定位模块的 HTTP 接口。将下文地址中的 IP 占位符替换为模块 IP 地址。除非接口另有说明，响应中的 code 为 200 且 message 为 success 表示请求已成功处理。

<Warning>
  更新程序、部署或切换地图、重启程序和重启相机会中断定位服务。请在车辆停止且具备恢复条件时调用这些接口。
</Warning>

## 程序和设备信息

### 更新主程序

地址：<code>http://&lt;ip&gt;:9998/tool/rob/quiet_update</code>

<ParamField body="url" type="string" required>
  主程序升级包的可访问 URL。
</ParamField>

### 设置相机高度

地址：<code>http://&lt;ip&gt;:9998/tool/rob/setHeightRequest</code>

<ParamField body="height" type="number" required>
  相机高度。
</ParamField>

### 查询程序版本

地址：<code>http://&lt;ip&gt;:9998/tool/rob/getRobotVersion</code>

该接口不需要请求参数。

<ResponseField name="code" type="integer" required>
  响应状态码。
</ResponseField>

<ResponseField name="data" type="string">
  程序版本，例如 version 12313。
</ResponseField>

### 查询车辆信息

地址：<code>http://&lt;ip&gt;:9998/tool/rob/getVehicle</code>

该接口不需要请求参数。

<ResponseField name="data" type="object" required>
  车辆信息对象。
</ResponseField>

<ResponseField name="vehicleId" type="string">
  车辆 UUID。
</ResponseField>

<ResponseField name="mapId" type="string">
  当前地图。
</ResponseField>

<ResponseField name="vehicleType" type="string">
  车辆类型。
</ResponseField>

### 查询当前位姿

地址：<code>http://&lt;ip&gt;:9998/tool/rob/getAGVPose</code>

该接口不需要请求参数。

<ResponseField name="pose" type="object" required>
  当前位姿。
</ResponseField>

<ResponseField name="x" type="number">
  X 坐标。
</ResponseField>

<ResponseField name="y" type="number">
  Y 坐标。
</ResponseField>

<ResponseField name="yaw" type="number">
  航向角。
</ResponseField>

### 重定位

地址：<code>http://&lt;ip&gt;:9998/tool/rob/relocate</code>

请求头：Content-Type: application/json

<ParamField body="x" type="number" required>
  X 坐标，单位为米。
</ParamField>

<ParamField body="y" type="number" required>
  Y 坐标，单位为米。
</ParamField>

<ParamField body="angle" type="number" required>
  姿态角。
</ParamField>

<ParamField body="noisyX" type="number" required>
  X 方向噪声，单位为米。
</ParamField>

<ParamField body="noisyY" type="number" required>
  Y 方向噪声，单位为米。
</ParamField>

<ParamField body="noisyAngle" type="number" required>
  角度噪声。
</ParamField>

## 地图管理

### 下载当前运行地图包

地址：<code>http://&lt;ip&gt;:9998/tool/rob/getRunMapZip</code>

请求头：Content-Type: application/json

<ParamField body="Name" type="string" required>
  地图名称。
</ParamField>

响应为 ZIP 文件，内容类型为 application/zip。

### 部署地图

地址：<code>http://&lt;ip&gt;:9998/tool/rob/deployRunMap</code>

<ParamField body="id" type="string" required>
  地图名称。
</ParamField>

### 应用地图

地址：<code>http://&lt;ip&gt;:9998/tool/rob/changeCurrentRunMap</code>

<ParamField body="name" type="string" required>
  地图名称。
</ParamField>

### 开始建图

地址：<code>http://&lt;ip&gt;:9998/tool/rob/startMapping</code>

<ParamField body="name" type="string" required>
  地图名称。
</ParamField>

### 结束建图

地址：<code>http://&lt;ip&gt;:9998/tool/rob/stopMapping</code>

该接口不需要请求参数。

### 开始或结束增量建图

地址：<code>http://&lt;ip&gt;:9998/tool/rob/sendCMD</code>

| 操作 | cmd | attach.sub_command | attach.param |
| --- | --- | --- | --- |
| 开始增量建图 | 904 | 0 | mapname |
| 结束增量建图 | 905 | 0 | mapname |

### 获取地图列表

推荐使用本机地图列表接口：

<code>http://&lt;ip&gt;:9998/tool/rob/getRunMapLists</code>

该接口不需要请求参数，响应中的 run_maps 为地图名数组。

<Note>
  后台地图列表接口 <code>http://&lt;ip&gt;:9998/tool/editor/getLocalMaps</code> 已标注为不推荐使用。
</Note>

### 上传地图包到相机

地址：<code>http://&lt;ip&gt;:9998/tool/rob/uploadMapZip2RunMaps</code>

<ParamField body="url" type="string" required>
  地图包的可访问 URL。
</ParamField>

### 下载地图

地址：<code>http://&lt;ip&gt;:9998/tool/rob/downloadVisiomWithLaserMap</code>

<ParamField body="url" type="string" required>
  地图包的可访问 URL。
</ParamField>

## Truy xuất thông tin toàn cầu

### Cập nhật chương trình chính

URL yêu cầu

`http://<ip>:9998/tool/rob/quiet_update`

Yêu cầu Ví dụ:

```json
{

"url":"http://192.168.2.179:8080/hontai_2.1.20211114.zip"

}
```

### Đặt chiều cao của Camera

URL yêu cầu `http://<ip>:9998/tool/rob/setHeightRequest` Ví dụ:

```json
{

"height": 1.3

}
```

Ví dụ phản hồi:

```json
{

"code":200, "message":"success",

}
```

- 1\. ** Tái định cư (Trực tuyến)**

URL yêu cầu

`http://<ip>:9998/tool/rob/relocate`

Thông số yêu cầu:

Loại nội dung: ứng dụng/json

|  |  |  |
| --- | --- | --- |
| Bắt buộc | Thông số | Mô tả |
| Có | x | Vị trí X (mét) |
| Có | y | Vị trí Y (mét) |
| Có | góc | – |
| Có | ồn àoX | Ồn ào (mét) |
| Có | ồn àoY | Ồn ào (mét) |
| Có | ồn àoAngle | Ồn ào (mét) |

Ví dụ yêu cầu: bỏ qua

Ví dụ phản hồi:

```json
{

"code":200, "message":"success",

}
```

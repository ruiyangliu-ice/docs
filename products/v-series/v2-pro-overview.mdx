V2 Pro 是一款面向移动机器人与人工叉车定位场景的导航控制器，融合顶视相机、2D LiDAR 和 IMU 数据，支持环境建图、定位与重定位。

完整产品资料可在[下载中心](/resources/downloads/specifications-and-user-guides)获取。
</Info>

## 产品概述

### 工作方式

控制器内置计算平台，无需额外的工控机。机器人在目标区域运行时，顶视相机采集环境特征，2D LiDAR 提供平面轮廓，IMU 提供运动数据；控制器融合这些数据以构建地图并输出定位结果。

### 主要特性

- 融合 `108° × 57°` 顶视相机、`270°` 扫描范围的 2D LiDAR 与 IMU。
- 配备 `6 TOPS` NPU 算力，支持通过可视化界面建图和管理地图。
- 集成 Wi-Fi 6 和音频输出，可用于人工叉车的定位数据传输、语音提示和告警。
- 支持移动机器人、AGV 和人工叉车的室内定位与导航集成。

### 典型应用

- **厂区物流**：仓库、月台和车间等区域的机器人定位与导航。
- **人工叉车升级**：为既有叉车加装定位能力，并通过上层系统接入位置数据。

### 接口定义

控制器提供电源、以太网、CAN、I/O 和音频输出接口。

<Frame caption="V2 Pro 接口位置">
  <img src="/images/legacy/24724a5b-15d1-4717-924b-26325b7ff494/e9bb4b68-b23a-45f3-873c-54beb31f17dc.avif" alt="V2 Pro 的以太网、CAN、I/O、电源和音频接口位置" />
</Frame>

| 端口 | 定义 | 说明 |
| --- | --- | --- |
| CN6 | 以太网 | RJ45 母座 |
| CN7 | CAN | 粉色：CAN_H；浅绿色：CAN_L；黑色：GND |
| CN8 | I/O | 4 路 I/O（2 路输入、2 路输出）。详见下方引脚定义。 |
| CN9 | 电源 | 24 VDC / 3 A；1：正极；G：负极 |
| CN10 | 音频输出 | 3.5 mm 音频插头 |

#### CN8 I/O 引脚定义

| 引脚 | 信号 | 引脚 | 信号 |
| --- | --- | --- | --- |
| 1 | IN1+ | 2 | IN1- |
| 3 | IN2+ | 4 | IN2- |
| 5 | OUT1+ | 6 | OUT1- |
| 7 | OUT2+ | 8 | OUT2- |

- 输入：5–24 V 高电平有效。
- 输出：推挽输出，支持 24 V / 100 mA 驱动。

### 无线通信模块

控制器配备 `2T2R` 无线网卡，支持 MU-MIMO 和双流传输。Wi-Fi 工作于 `2.400–2.4835 GHz` ISM 频段；实际吞吐量受网络配置、现场干扰和接入设备影响。

| 参数 | 规格 |
| --- | --- |
| Wi-Fi 标准 | IEEE 802.11ax/ac/a/b/g/n（2T2R） |
| 蓝牙版本 | Bluetooth 5.3、5.0、4.2、4.1、4.0 LE、3.0+HS、2.1+EDR |
| Wi-Fi 数据速率 | 802.11b：11 Mbps；802.11a/g：54 Mbps；802.11n：MCS0–15；802.11ac：MCS0–9；802.11ax：HE0–11 |
| 蓝牙数据速率 | 1 Mbps、2 Mbps，最高 3 Mbps |
| 工作频段 | 2.400–2.4835 GHz ISM 频段；不支持 5 GHz 频段 |

| 序号 | 内容 |

### 连接并验证设备

<Frame caption="V2 Pro 连接测试示意">
  <img src="/images/legacy/dfd4c3be-816c-4ffc-81ac-13141e3c2644/9d8858e6-85a0-492f-a353-4e5adf1f81a2.avif" alt="V2 Pro 电源和以太网连接示意图" />
</Frame>

<Steps>
  <Step title="连接电源">
    将电源线接入控制器的 DC 电源接口，并连接 `24 V / 3 A` 直流电源。蓝色指示灯缓慢闪烁表示设备已正常上电。

    若指示灯不亮或红灯常亮，请检查电源接口和电源适配器。
  </Step>
  <Step title="配置网络">
    使用六类网线将控制器以太网接口与计算机直接连接。设备出厂默认 IP 为 `192.168.100.201`；将计算机设置为同一网段，并允许防火墙放行设备通信。

    在局域网浏览器中访问 `http://192.168.100.201:9998`，可免密登录 Web 界面。
  </Step>
  <Step title="检查数据流">
    启动配套上位机软件，确认顶视相机图像清晰，且没有花屏或明显卡顿。若检测到多个 IP 地址，请排查局域网 IP 冲突并选择正确的设备地址。
  </Step>
</Steps>

### 界面说明

<Frame caption="V2 Pro 上位机界面">
  <img src="/images/legacy/4e6942a8-55ab-4030-9609-4c1392a560fa/69362392-66e2-4bd2-8fc3-72e4a3e1efbe.avif" alt="标注菜单栏、轮廓图、顶视图和定位状态的 V2 Pro 上位机界面" />
</Frame>

1. **红色区域**：功能菜单，包括建图、重定位和地图管理。
2. **蓝色区域**：已加载的 2D 轮廓图和车辆定位图标。
3. **绿色区域**：顶视相机实时 RGB 图像。
4. **黄色区域**：重定位信号强度、车辆位置和速度，以及鼠标位置坐标。信号强度依次为绿色（优）、蓝色（良好）和红色（差）。

### 创建地图

<Steps>
  <Step title="开始建图">
    在菜单栏选择 **开始建图**。

    <Frame>
      <img src="/images/legacy/edcaac72-adf1-4432-ba27-9b7323cde695/8cdd9468-9cb7-437d-a823-4a0cbffae053.avif" alt="V2 Pro 上位机中的开始建图入口" />
    </Frame>
  </Step>
  <Step title="输入地图名称">
    地图名称仅可包含字母、数字和下划线，且不能以数字开头。
  </Step>
  <Step title="采集地图数据">
    建图时，车辆线速度不得超过 `0.5 m/s`，角速度不得超过 `20°/s`。界面会显示激光轮廓和顶视相机图像。

    <Frame caption="建图界面">
      <img src="/images/legacy/2568abac-69b0-4e56-af86-53575c614918/12edb330-0198-49f8-a1de-cbcc05192b4a.avif" alt="V2 Pro 建图界面中的激光轮廓和顶视相机图像" />
    </Frame>

    保持低速、匀速行驶，建议速度小于 `0.2 m/s`。尽量直行和直角转弯，避免重复路线和随机抖动；同向轨迹保留约三分之一的重叠区域，平行轨迹间距约 `2 m`，以覆盖目标区域或车辆行驶路线。

    <Frame caption="建议的建图行驶路径">
      <img src="/images/legacy/7f2923e9-c17c-47a0-8720-9de448d0a851/045b2eb3-7aca-4ae2-8693-660381d3301e.avif" alt="V2 Pro 建图时建议采用的平行行驶路径" />
    </Frame>
  </Step>
  <Step title="结束建图">
    完成区域扫描后，在菜单栏选择 **结束建图**。处理时间取决于扫描区域的大小。
  </Step>
</Steps>

### 部署和应用地图

<Steps>
  <Step title="部署后台地图">
    打开 **地图管理**，在后台地图列表中找到新建地图并选择 **部署**。需要使用的每张地图都必须部署到主机地图；后台地图支持修改、导出和删除。

    <Frame caption="后台地图管理">
      <img src="/images/legacy/60beebd0-634a-4145-907c-e553c16a5db1/82af1a8d-d514-4304-89d0-439e413314b3.avif" alt="V2 Pro 后台地图管理界面" />
    </Frame>
  </Step>
  <Step title="应用主机地图">
    在 **主机地图** 中选择 **应用**。应用后必须重启控制器，地图才会生效。主机地图支持导出和删除。

    <Frame caption="主机地图管理">
      <img src="/images/legacy/22d53108-6c4c-4352-afde-cea020bcf3dc/60add4a5-7fb8-403b-89fa-52c52a02ac2f.avif" alt="V2 Pro 主机地图管理界面" />
    </Frame>
  </Step>
</Steps>

### 重定位与参数配置

<Frame caption="重定位界面">
  <img src="/images/legacy/a2c8e5ae-7696-4335-aac3-d104f19e5661/deb0504b-1722-4266-9085-373b38eaf7b1.avif" alt="V2 Pro 基于激光点云和轮廓图进行重定位的界面" />

在 **重定位** 界面中，拖动鼠标为车辆指定大致位姿，并根据激光点云与轮廓图的匹配提示完成重定位。成功后，定位超时提示会消失。

在参数配置中，检查并设置底盘定位参数，以及 LiDAR 与顶视相机的外参标定值。

## 故障排查

| 现象 | 建议操作 |
| --- | --- |
| 打开设备无响应 | 检查防火墙规则，确认没有拦截控制器通信。 |
| 上位机检测到多个 IP 地址 | 排查局域网 IP 冲突，并手动选择正确的设备 IP 地址。 |
| 数据流不稳定 | 使用质量合格的千兆网线；低速网线可能导致初始数据传输不稳定。 |
| 上位机启动异常 | 避免将软件安装在受系统权限严格限制的目录；选择具有读写权限的安装目录。 |

## 安全与合规

### 激光安全

<Warning>
设备工作时会发射不可见激光。请避免使眼睛暴露于激光发射区域，并保持设备上的安全标签清晰、完整。
</Warning>

- 本产品的激光符合 Class 1 安全等级。
- 按照 EN 60825 的要求，在正常使用条件下，产品发射的激光不会对人体造成危害。
- 安装、维护或异常排查时，请遵守现场的激光安全规程；不得擅自拆解或改装光学相关部件。

</Frame>

### 内含锂电池

<Warning>
设备内含锂电池。电池受损、短路、挤压、刺穿或接近火源可能引发起火等危险。
</Warning>

- 请勿自行拆卸、改装或更换内置电池。
- 避免使设备受到剧烈撞击、挤压、刺穿或高温、火源影响。
- 运输、报废或回收设备时，请遵守当地关于含锂电池设备的规定。

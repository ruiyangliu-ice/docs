H 系列面向高精度三维识别、定位与抓取任务。H3310 采用双目编码结构光方案，在 300–600 mm 工作距离内输出 RGB、深度与幅度数据，适用于自动化生产、物流和机器人末端感知。

## H3310 规格

| 参数 | H3310 |
| --- | --- |
| 工作原理 | 双目编码结构光 |
| 深度分辨率 | 1280 × 1024 |
| RGB 分辨率 | 1920 × 1080 |
| 数据采集时间 | 约 0.8 s |
| 输出数据 | 深度图、RGB 图、幅度图 |
| 工作范围 | 300–600 mm |
| 视场角 | 55° × 45° |
| 测量精度 | ±0.1 mm（@ 350 mm） |

| 重量 | 400 g |

| SDK | C/C++ |
| 第三方支持 | ROS、ROS2 |

| 散热方式 | 被动散热 |

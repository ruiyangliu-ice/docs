## 产品概述

S10 是一款工业级 dToF RGB-D 相机，可提供 0.3–8 m 深度数据。设备支持 RGB 与深度图的时空对齐，可输出深度图、IR 幅值图、RGB 图、点云和避障算法结果。

相机集成 SoC，提供以太网与 I/O 接口。2 路数字输入和 2 路数字输出可用于切换 4 组避障区域，并输出停止、减速或安全信号。

### 术语

| 术语 | 说明 |
| --- | --- |
| RGB-D 相机 | 同时输出 RGB 图像和深度图，并保持时间与空间同步。 |
| 深度图 | 每个像素的数值表示其到相机的距离。 |
| FOV | 视场角，包含水平（HFOV）、垂直（VFOV）和对角线（DFOV）视场角。 |
| IR 相机 | 对近红外波段成像的相机或模组。 |
| ToF | 通过测量光飞行时间获取距离的技术。 |
| dToF | 直接飞行时间测距技术。 |
| SoC | 用于图像、深度和应用算法处理的系统级芯片。 |

### 工作原理

S10 基于 dToF 技术测距：相机发射激光并检测目标反射光的到达时间，以计算目标距离。设备同时集成 RGB 模组，并在相机内部完成 RGB 与深度数据对齐。

### 功能特性

- 紧凑型 dToF 相机，支持中距离深度感知。
- 集成计算能力，可输出障碍物检测结果。
- 支持 100 kLux 环境光条件下的使用。
- 支持深度、RGB、IR 幅值和点云数据输出。

## 首次使用

<Steps>
  <Step title="连接电源和网络">
    使用 24 VDC / 2 A 电源为相机供电。蓝色指示灯慢闪表示上电正常。使用六类网线将相机连接至电脑。
  </Step>
  <Step title="配置网络">
    相机默认 IP 地址为 `192.168.100.82`。将电脑设置为相同网段，并确保防火墙未阻断相机通信。
  </Step>
  <Step title="打开相机">
    在配套上位机中打开相机，检查图像输出。上位机软件可向 MRDVS 销售或技术支持获取。
  </Step>
</Steps>

### 常见问题

| 问题 | 排查建议 |
| --- | --- |
| 点击打开相机无响应 | 检查防火墙是否阻断通信，并确认电脑与相机在同一网段。 |
| 上位机检测到多个 IP | 排除局域网中的 IP 地址冲突，并选择正确设备。 |
| 数据流不稳定 | 建议使用千兆网线；百兆网线可能导致初始数据不稳定。 |
| 软件启动异常 | 检查默认安装目录的系统权限。 |

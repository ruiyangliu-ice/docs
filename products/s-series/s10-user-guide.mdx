## 1.前言

### 1.1 文档目的与范围

本文档介绍了S10相机的产品规格，以及供开发者了解和使用相关产品的部分设计细节。

### 1.2术语

| RGBD相机 | 相机支持同时输出RGB图像和depth图像，并保证时间和空间同步 |
| --- | --- |
| Depth | 深度视频流与彩色或者黑白视频流类似，每个像素都有一个值代表距离摄像机的距离，即‘深度’ |
| FOV | 视场角，用于描述相机成像给定场景的角度范围，主要有水平视场角(HFOV)、垂直视场角(VFOV)和对角线视场角(DFOV) |
| IR camera | 红外相机，或红外摄像头，本产品中主要针对近红外波段成像 |
| RGB camera | 彩色相机，或彩色摄像头 |
| Depth camera | 深度相机，其中深度成像模组一般由点阵投射器、红外相机以及深度计算处理器组成，彩色成像模组一般指彩色相机 |
| Soc | 系统级芯片，用于处理图像数据，深度结算，应用算法运算等功能 |
| DTOF | 基于飞行时间测量原理的距离传感技术 |

### 1.3工作原理

本产品是基于DTOF的技术，它具有测距范围广、精度良好、成本低、抗干扰性好等优点,数据更稳定更可靠的3D深度相机。同时整机集成RGB模块，实现彩色图像的获取。在相机内部可以实现RGB和depth图像的对齐功能。

## 2.产品概述

该产品是一款，小体积，高性能，高性价比的工业级RGBD相机。可提供0.3m-8m的高性能深度数据。相机内部实现RGB图和depth图，空间和时间上的对齐功能。该产品自带SOC，可以输出深度图，强度图，RGB图，点云图和避障算法结果。接口上不仅支持以太网接口，同时该相机支持通过2路IO输出，2路IO输入，实现4组避障区域的切换和急停信号、减速信号、安全三种信号的输出。

### 2.2 功能特性

- DTOF功耗低，并且体积小巧

- 内置算力，支持输出障碍物检测结果，实现即插即用

- 环境适应性强：可抗100KLUX阳光干扰，从黑暗到强光，室内到室外均适用。

- 测量范围广：可覆盖0.3-8米距离范围。

## 6.快速使用指南

### 6.1连接测试

<Frame>
  <img src="/images/legacy/cb764c04-73c2-4cdd-9381-98fe1c201737/24cb2351-298f-4381-8170-c34cf1444269.avif" alt="连接测试步骤图" />
</Frame>

1.  相机DC电源接24V/2A，供电。电源指示灯蓝灯慢闪，表示上电正常。

2.  相机通过六类网线和电脑连接。

3.  相机出厂默认IP192.168.100.82。

4.  设置电脑ip和相机同一网段，关闭电脑防火墙。

5.  运行配套上位机软件，获取相机图像。

6.  上位机软件联系迈尔微视销售或者技术支持获取。

### 6.2 IO接线说明

| 接口 | 属性 | 备注 |
| --- | --- | --- |
| IN1 | Input | 离散量输入信号，高有效，支持 24V/开接入 |
| IN2 | Input | 离散量输入信号，高有效，支持 24V/开接入 |
| OUT1 | Output | 离散量输出信号，地/开信号，支持 24V 上拉 |
| OUT2 | Output | 离散量输出信号，地/开信号，支持 24V 上拉 |
| COM | GND | 离散量输出信号共地点，IN1和IN_COM1成对使用 |

## 8.常见问题QA

| 序号 | 问题 | 说明 |
| --- | --- | --- |
| 1 | 点击打开相机没反应 | 请看下防火墙是否关闭，需要关闭防火墙 |
| 2 | 上位机获取到多个IP | 局域网内有相同IP被相机获取到，需要选择需要的IP打开 |
| 3 | 打开相机数据不稳定 | 相机建议使用千兆网线，使用百兆网线刚开始的数据会不稳定 |
| 4 | 软件安装位置 | 如安装默认C盘，可能有权限设置 |

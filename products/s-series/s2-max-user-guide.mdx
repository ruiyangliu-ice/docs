## 产品概述

S2 Max 是一款采用千兆以太网接口、24 VDC 供电的 ToF RGB-D 相机。设备集成约 1 TOPS 算力，可输出深度、RGB 和点云数据，并支持避障、库位检测和体积检测等应用。

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-s2-max-camera.avif" alt="S2 Max 相机" />
</Frame>

### 使用要求

- 请勿撕毁产品标签、拆解产品或修改内部文件。
- 避免在强磁环境和超过 100 kLux 的直射强光环境中使用。
- 清洁外壳时，使用柔软无尘布蘸少量纯净水或酒精擦拭。
- 请保留原始包装，并在运输时优先使用原始包装。

## 测距原理与规格

### 测距原理

S2 Max 采用 ToF 测距：相机发射光信号并测量其经目标反射后返回的时间，据此计算目标到相机的距离。ToF 技术可分为脉冲 ToF（P-ToF）和连续波调制 ToF（CW-ToF）；S2 Max 通过相应的深度计算流程构建深度场景。

## 输出、探测与结构

### 输出数据

默认情况下，相机输出深度图、对应的强度图和由深度数据生成的点云。深度图采用隔帧高动态范围（HDR）方式扩展测量范围：系统结合不同积分时间的深度数据，为每个像素选择信号较强的数据并进行融合。

### 探测范围

相机的水平和垂直探测角度分别为 81° 和 61°。光功率和目标反射率会影响可用的近、远测距范围；对于 10%–90% 反射率目标，标称范围为 0.2–5 m。超过 5 m 后，较低的反射信号可能使环境光引起的测距偏差增大。

## 准确度、精度与多机干扰

### 准确度与精度

测量准确度依赖于标定，用于修正测量值与实际距离之间的固定偏差。安装条件、目标反射率和近旁高反射物体会引入多路径和眩光，从而影响结果。

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-calibration-setup.avif" alt="S2 Max 标定示意图" />
</Frame>

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-camera-installation-light-paths.avif" alt="S2 Max 高反射环境下的多路径干扰示意图" />
</Frame>

精度通常以单点波动（1σ）描述。ADC 量化、信噪比和系统噪声都会影响测距精度；较低信号强度通常会增大测量波动。可通过调整积分时间等参数改善信号质量，但应在实际场景中验证。

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-precision-and-signal-strength.avif" alt="S2 Max 测量精度与信号强度关系图" />
</Frame>

### 多机干扰

两台相机平行安装并观测同一方向时，可能接收到彼此的反射光，从而增加深度数据误差。建议使相机之间保持一定夹角；若必须平行安装，请联系 MRDVS 评估定制方案。

## SDK 与上位机

### SDK

SDK 用于初始化和关闭相机、获取数据及进行跨平台开发。

| 项目 | 说明 |
| --- | --- |
| 操作系统 | Linux、Windows |
| 编程语言 | C/C++ |
| 数据格式 | 16 位无符号整型 |
| 使用方式 | `.so`、`.dll` 或集成 SDK 源码 |

### 数据输出顺序

SDK 按帧输出深度图和强度图，并以一维数组存储。数据从图像左上角开始，按从左到右、从上到下的顺序排列；像素 `(x, y)` 的行优先索引为 `y × width + x`。

深度图的每个像素使用 16 位无符号整数表示目标到相机所在平面的距离；强度图的每个像素使用 16 位无符号整数表示返回信号强度。

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-sdk-output-order.avif" alt="S2 Max SDK 数据输出顺序" />
</Frame>

### IP 地址

相机默认 IP 地址为 `192.168.100.82`。如需改为静态 IP 或 DHCP，请使用 MRDVS 提供的演示工具。

### 演示软件

#### 配置网络

先完成电源和网线连接。将电脑设置到 `192.168.100.x` 网段（不要使用 `192.168.100.82`），子网掩码为 `255.255.255.0`。例如，可将电脑 IP 设置为 `192.168.100.20`。

#### 打开相机

安装并启动 LxCameraViewer。单台相机连接时，软件会自动搜索、打开相机并开始采集。首次运行时，部分系统需要允许软件通过防火墙。

软件显示相机 IP、ID、MAC、序列号、软件版本和算法版本等信息。勾选深度图、强度图、RGB 或点云，即可在中间区域查看相应数据。

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-lxcameraviewer-image-display.avif" alt="S2 Max LxCameraViewer 图像显示界面" />
</Frame>

#### 保存数据

点击 **保存** 后，软件会保存已勾选的数据，默认目录为安装路径下的 `./data`。终端会输出保存路径，例如 `./data/相机ID`。

- `_a`：强度图
- `_d`：深度图
- `_r`：RGB（MONO）图
- `.pcd`：点云

深度图和强度图使用 PGM 格式，RGB 使用 PNG 格式，点云使用 PCD 格式。

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-saved-image-files.avif" alt="S2 Max 图像保存结果示例" />
</Frame>

#### 修改相机 IP

输入目标 IP、子网掩码和网关后确认修改。例如，IP 为 `192.168.10.199` 时，可设置子网掩码为 `255.255.255.0`，网关为 `192.168.10.1`。修改后为相机断电并重新上电，使配置生效。

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-change-the-camera-ip-address.avif" alt="S2 Max 修改 IP 地址界面" />
</Frame>

## 安全与常见问题

### 激光安全

<Frame>
  <img src="/images/products/s-series-s2-max-user-guide-laser-safety-class.avif" alt="S2 Max 激光安全标签" />
</Frame>

请保持安全标签完整、清晰，并按标签和产品资料中的安全要求使用设备。

### 正确操作

- 接通电源前完成接线，并确认正负极未接反；反接可能损坏产品。
- 工作温度不得超出规格范围。
- 建议安装在导热良好的金属支架上，以改善散热并保持性能稳定。

### 常见问题

| 问题 | 排查建议 |
| --- | --- |
| 点击打开相机无响应 | 检查防火墙设置，并确认电脑与相机处于同一网段。 |
| 上位机检测到多个 IP | 排除局域网中的 IP 地址冲突，并选择正确设备。 |
| 数据流不稳定 | 建议使用千兆网线；百兆网线可能导致初始数据不稳定。 |
| 软件启动异常 | 检查默认安装目录的系统权限。 |

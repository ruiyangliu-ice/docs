## 产品概述

M4 是一款采用千兆以太网通信、24 VDC 供电的 ToF RGB-D 相机。产品型号包括 M4 Lite 和 M4。

<Frame>
  <img src="/images/legacy/e882097c-c641-4cc8-8d4a-1e3856985f4c/1752f961-5a67-4a8a-b7d1-3bd19dae0f11.avif" alt="M4 相机外观" />
</Frame>

### 产品特点

- 集成 1 TOPS 计算能力，可运行应用算法。
- 输出 RGB-D 数据，适用于避障、对接、定位、软包抓取、库位检测和体积测量等任务。
- 提供千兆以太网和 CAN 接口。

### 使用要求

- 请勿移除产品标签、拆解设备或修改内部文件。
- 避免在强磁环境和超过 100 kLux 的阳光干扰环境中使用。
- 清洁外壳时，使用蘸有少量纯净水或酒精的柔软无尘布。
- 运输时尽量使用原始包装。

## 工作原理与规格

### ToF 测距原理

M4 使用飞行时间（ToF）法测量发射光经目标反射后返回相机的时间，从而计算目标距离。相机通过主动光成像获取深度信息。

### 工作范围与反射率

M4 的水平、垂直视场角分别为 81° 和 61°。实际范围受目标反射率、环境光和安装条件影响。对于低反射率或远距离目标，返回信号减弱，测距偏差可能增大。

避免将相机同时朝向近距离地面和侧墙；地面杂散光与墙面多径反射可能影响测距。

## 精度与多机部署

### 准确度和精度

标定用于消除固定系统误差。实际测量结果还会受安装方式、目标反射率、多径反射、眩光、环境光和信号强度影响。

<Columns cols={2}>
  <Frame><img src="/images/legacy/f56d3e54-eb77-4bdb-9576-65f9aeb7a628/67a4d596-b169-43c0-93b0-dfa2ef85c421.avif" alt="M4 距离标定示意图" /></Frame>
  <Frame><img src="/images/legacy/be0eb189-bf98-4c78-b2dc-3a4e8a356124/04671b63-d54f-4cac-8774-4afb08cccbc2.avif" alt="高反射物体造成多路径干扰的示意图" /></Frame>
</Columns>

单点精度通常用 1σ 波动描述。信号较弱时，量化噪声和系统噪声会更明显；增加积分时间通常有助于提高信噪比，但应结合应用的帧率要求进行设置。

<Frame>
  <img src="/images/legacy/16312574-c0be-48c1-81b8-0cfc40ff06ae/74264eea-5a19-48a4-8520-84410a78262c.avif" alt="测量精度与信号强度的关系示意图" />
</Frame>

### 多机干扰

多台 M4 平行安装并观测同一方向时，可能接收到其他相机的主动光，从而增加深度测量误差。建议使相机之间保持一定夹角；需要平行部署时，请进行现场验证。

## 软件与数据

### SDK

SDK 可用于初始化、启动、取数和关闭相机，并提供共享库、动态链接库或源代码集成方式。

| 项目 | 支持情况 |
| --- | --- |
| 操作系统 | Linux、Windows |
| 编程语言 | C/C++ |
| 数据类型 | 16 位无符号整数 |
| 集成方式 | `.so`、`.dll` 或 SDK 源码 |

SDK 按帧将深度图和强度图展平为一维数组，以图像左上角为起点，按从左到右、从上到下的行优先顺序排列。每个深度和强度像素均为 16 位无符号整数。

</Frame>

### LxCameraViewer

默认 IP 地址为 `192.168.100.82`。使用上位机修改 IP 或启用 DHCP 前，先将计算机配置到同一网段。首次运行时，按系统提示允许软件访问网络。

<Frame>
  <img src="/images/legacy/1b245f57-ab6e-4a00-bfd0-810d28877644/afbf32b7-d850-487a-b003-d739ab1ce912.avif" alt="LxCameraViewer 的 M4 图像显示界面" />
</Frame>

勾选所需数据类型后点击保存，文件默认保存在软件安装目录的 `./data` 文件夹。强度图和深度图使用 PGM 格式，RGB 图使用 PNG 格式，点云使用 PCD 格式。

<Frame>
  <img src="/images/legacy/16abb878-910c-4b62-b84c-c1100c0eead4/0d8b3485-3661-425c-83e6-2dd524c8c4e6.avif" alt="LxCameraViewer 的图像与点云保存结果" />
</Frame>

修改 IP 时，填写目标 IP、同网段子网掩码与网关，确认后重启相机使设置生效。

<Frame>
  <img src="/images/legacy/aa5ef83b-5b0b-488f-a9b3-1ac04d897693/0578871e-db36-4ee3-abd1-33795a4860fe.avif" alt="LxCameraViewer 的 IP 地址修改界面" />
</Frame>

SDK 提供中值滤波、众数滤波和飞点去除等处理方法。M4 的标准测量范围为 0.3–5 m。

## 产品概述

M4 Mega 是一款采用千兆以太网通信、24 VDC 供电的 iToF RGB-D 相机，支持 CAN 和 I/O。相机可输出空间与时间对齐的 RGB、深度与幅度数据，并可通过 CAN 输出托盘识别结果的结构化数据。

<Frame>
  <img src="/images/products/m-series-m4-mega-user-guide-m4-mega-camera-overview.avif" alt="M4 Mega 相机外观" />
</Frame>

### 产品特点

- 集成 1 TOPS 计算能力，可运行应用算法。
- 支持 RGB-D 数据输出，适用于避障、对接、定位、软包抓取、库位检测和体积测量等任务。
- 提供千兆以太网、CAN、4 路 I/O 和 IP67 防护。

### 使用要求

- 请勿移除产品标签、拆解设备或修改内部文件。
- 避免在强磁环境和超过 100 kLux 的阳光干扰环境中使用。
- 清洁外壳时，使用蘸有少量纯净水或酒精的柔软无尘布。
- 运输时尽量使用原始包装。

M4 Mega 使用飞行时间（ToF）法测量发射光经目标反射后返回相机的时间，从而计算目标距离。

## 触发功能

硬触发和软件触发仅在对应触发模式下有效，且均与避障模式互斥。切换触发模式前，请确认避障算法已关闭并停止取流。

### 硬件触发

硬触发模式下，相机等待 `IN_1` 上的有效脉冲；每收到一个有效触发，开始一次曝光或按设定连续曝光。触发输入电压为 **3.3–24 V**，驱动能力应不低于 **10 mA**。建议在触发输入端加入去耦措施以降低误触发风险。

在 LxCameraViewer 中依次选择：**打开相机** → **确认避障算法关闭** → **停流** → **基础工具** → **采集设置** → **触发模式：硬触发** → **设置参数** → **启流**。

| 参数 | 说明 | 范围或建议 |
| --- | --- | --- |
| 滤波时间 | 收到信号后等待并复检触发状态，用于滤除噪声。 | 通常 < 5 ms |
| 最小时间间隔 | 连续帧之间的时间间隔。 | 70,000–100,000,000 μs；必须大于积分时间 |
| 触发延时 | 复检触发后到第一帧曝光前的延时。 | 0–100,000,000 μs |
| 单次触发帧数 | 单次触发后连续输出的帧数。 | 1–200 |

例如，滤波时间为 `1000 μs`、最小时间间隔为 `70000 μs`、触发延时为 `2000 μs`、单次触发帧数为 `10` 时，相机在复检后延时 2 ms 输出第一帧，之后每隔 70 ms 输出一帧，共输出 10 帧。此期间后续触发会被忽略。

### 软件触发

软件触发模式下，相机等待上位机的有效触发命令。操作路径为：**打开相机** → **确认避障算法关闭** → **停流** → **基础工具** → **采集设置** → **触发模式：软触发** → **设置参数** → **启流** → **发送软触发**。

滤波时间、最小时间间隔、触发延时和单次触发帧数的含义与硬件触发一致。设置单次触发帧数为 `1` 时，最小时间间隔不生效。

## 精度与多机部署

标定用于消除固定系统误差。实际测量结果还会受安装方式、目标反射率、多径反射、眩光、环境光和信号强度影响。

<Columns cols={2}>
  <Frame><img src="/images/products/m-series-m4-mega-user-guide-m4-mega-calibration-setup.avif" alt="M4 Mega 距离标定示意图" /></Frame>
  <Frame><img src="/images/products/m-series-m4-mega-user-guide-m4-mega-installation-and-reflection-paths.avif" alt="高反射物体造成多路径干扰的示意图" /></Frame>
</Columns>

单点精度通常用 1σ 波动描述。信号较弱时，量化噪声和系统噪声会更明显；增加积分时间通常有助于提高信噪比，但应结合应用的帧率要求进行设置。

<Frame>
  <img src="/images/products/m-series-m4-mega-user-guide-measurement-precision-and-signal-intensity.avif" alt="测量精度与信号强度的关系示意图" />
</Frame>

多台 M4 Mega 平行安装并观测同一方向时，可能接收到其他相机的主动光，从而增加深度测量误差。建议使相机之间保持一定夹角；需要平行部署时，请进行现场验证。

## 软件与数据

### SDK

SDK 可用于初始化、启动、取数和关闭相机，并提供共享库、动态链接库或源代码集成方式。

| 项目 | 支持情况 |
| --- | --- |
| 操作系统 | Linux、Windows |
| 编程语言 | C/C++ |
| 数据类型 | 16 位无符号整数 |
| 集成方式 | `.so`、`.dll` 或 SDK 源码 |

SDK 按帧将深度图和强度图展平为一维数组，以图像左上角为起点，按从左到右、从上到下的行优先顺序排列。

本指南帮助你完成 M4 Pro 的开箱检查、供电、网络配置和首次取流。

<Warning>
安装或连接线缆前，请断开电源。请勿遮挡镜头或触摸镜头表面。
</Warning>

<Steps>
  <Step title="检查包装内容">
    检查相机和配件是否完整、无运输损伤。如有缺失或损坏，请联系技术支持。

    <Frame>
      <img src="/images/external/c5fe2206c743bcceb51ead3a.avif" alt="M4 Pro 相机包装清单" />
    </Frame>
  </Step>
  <Step title="安装并连接相机">
    安装时保持镜头视野无遮挡。左图展示正确的安装状态，右图展示会遮挡镜头的错误状态。

    <Frame>
      <img src="/images/external/a2d017b8fff069857a6df0a3.avif" alt="M4 Pro 相机正确安装方式" />
    </Frame>

    <Frame>
      <img src="/images/external/84c19b4c11995706c8cddd81.avif" alt="M4 Pro 相机镜头被遮挡的错误安装方式" />
    </Frame>

    使用随附的 5-pin 线缆连接相机，并使用 **24 VDC / 3 A** 电源供电。蓝色电源指示灯以约 2 秒周期慢闪时，表示已正常上电。使用 Cat 6 或更高规格网线连接相机和计算机。

    <Frame>
      <img src="/images/legacy/2e3dae80-ce2f-4248-bcb2-5d07b6ab805a/c4c0a700-2858-4222-92c5-9ee61197ae57.avif" alt="M4 Pro 的供电与以太网连接方式" />
    </Frame>

    线缆信号定义请参阅 [M4 Pro 产品用户手册](/products/m-series/m4-pro-user-guide)。
  </Step>
  <Step title="配置计算机网络">
    M4 Pro 的默认 IP 地址为 `192.168.100.82`。将计算机以太网适配器设置为同一网段，例如 `192.168.100.100`，子网掩码设置为 `255.255.0.0`。

    在 Windows 中，依次打开 **控制面板** → **网络和 Internet** → **网络和共享中心** → **更改适配器设置** → **以太网** → **Internet 协议版本 4 (TCP/IPv4)**。

    <Frame>
      <img src="/images/external/276992291dbad945ddfdb9f7.avif" alt="Windows IPv4 地址设置界面" />
    </Frame>

    如果软件无法发现相机，检查本机防火墙是否阻止该网络连接。
  </Step>
  <Step title="安装 LxCameraViewer">
    LxCameraViewer 可查看图像、配置相机参数，并选择相机内置的应用算法。Windows 提供图形上位机；Linux 可通过示例程序和 RViz 查看数据。

    - [CameraSDK GitHub 仓库](https://github.com/Lanxin-MRDVS/CameraSDK)
    - [软件下载](https://zcnt5g3o2989.feishu.cn/wiki/Q8IYwGiGaiIvIyktiavczymanOe?from=from_copylink)
  </Step>
  <Step title="打开相机并验证数据">
    启动 LxCameraViewer 后，确认左侧设备列表中出现相机 ID。

    <Frame>
      <img src="/images/legacy/e00efd7e-06aa-4de8-9178-7e7e8e92c011/0629f2ec-63dc-4388-ac39-b78e306030df.avif" alt="LxCameraViewer 启动界面" />
    </Frame>

    若未显示设备，请先检查指示灯：正常供电时蓝灯约每 2 秒闪烁一次；常亮时请联系技术支持；不亮时检查 24 VDC / 3 A 供电和线缆。

    <Frame>
      <img src="/images/legacy/fcd6242d-8a7a-41e7-8cc0-bef2f10bec65/a0cf4ddc-5e81-4e85-85ee-3fef57c20c2e.avif" alt="LxCameraViewer 中的 M4 Pro 设备列表" />
    </Frame>

    选择设备并点击 **打开**。按钮变为 **关闭** 时表示已开始取流。默认输出深度图、强度图和点云；勾选对应选项可启用 RGB 图。

    <Frame>
      <img src="/images/legacy/e5741b74-091f-4bbe-81b8-374b66f6fd68/7fe62674-9ea9-4bd6-bb35-7db6bc2c7bdd.avif" alt="LxCameraViewer 中的图像流选择界面" />
    </Frame>

    在右侧展开参数面板，调整成像参数和工作模式。

    <Frame>
      <img src="/images/legacy/726a4493-2e26-4dfa-8458-c95c4ccac5d9/7c7194ea-18ac-434e-9ddb-a67d02000d23.avif" alt="LxCameraViewer 的相机参数设置面板" />
    </Frame>

    <Frame>
      <img src="/images/legacy/ed478a24-b097-4a66-98ca-ff842bca6042/c7d959c9-e656-4ba9-b0b8-ab74487456c6.avif" alt="LxCameraViewer 的应用算法与工作模式选择界面" />
    </Frame>

    有关完整的软件操作，请参阅 [LxCameraViewer 上位机软件使用手册](https://zcnt5g3o2989.feishu.cn/wiki/IHK1wiLy2iIccEkgZ3pc74jhnAf?from=from_copylink)。
  </Step>
</Steps>

## 使用注意事项

- 请勿使相机跌落、受撞击或承受其他机械应力。
- 请勿修改相机内部文件或拆卸设备；这可能导致永久性损坏。
- 长时间运行时外壳温度可能上升。相机采用被动散热，安装时可增大相机与金属安装板的接触面积。
- 镜头上的污渍或指纹会降低图像质量。
- 无法识别相机时，检查电源是否满足 24 VDC / 3 A，以及网线是否损坏。

## 安全说明

<Warning>
若设备有外观损坏，请勿上电。请勿自行拆机、改装或维修；避免让镜头正对阳光。
</Warning>
